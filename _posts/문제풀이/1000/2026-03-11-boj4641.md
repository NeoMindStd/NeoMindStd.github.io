# 풀이

## 문제

주어진 수열에서 어떤 수의 두 배가 같은 수열 안에 존재하는 경우의 수를 세는 문제입니다.

## 코드

```python
while True:
    s = input()
    if s == '-1': break
    
    s = list(map(int, s.split()))

    c = 0

    for no in s[:-1]:
        if s.count(no*2) > 0: c += 1

    print(c)
```

## 설명

각 원소 `x`에 대해 `2x`가 수열에 존재하는지 검사해 카운트합니다.
입력 끝 표식 `-1`에서 종료합니다.
