# 풀이
## 문제

입력 `N`에 대해
오른쪽 정렬된 별 삼각형을 출력하는 문제입니다.

## 코드

```python
a = int(input())

for i in range(a):
    for j in range(a-i-1):
        print(" ", end="")
    for j in range(a-i-1, a):
        print("*", end="")
    print()
```

## 설명

각 줄마다 앞 공백 `N-i-1`개, 별 `i+1`개를 출력해
오른쪽 정렬 형태를 만듭니다.
