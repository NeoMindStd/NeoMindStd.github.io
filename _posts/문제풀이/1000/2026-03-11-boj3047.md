# 풀이

## 문제

세 정수를 정렬한 뒤,
문자열 패턴(`ABC` 순열)에 맞춰 값을 출력하는 문제입니다.

## 코드

```python
abc = list(map(int, input().split()))
abc.sort()

s = input()

for i in range(3):
    print(abc[ord(s[i])-ord('A')], end=" ")
```

## 설명

정렬된 배열에서 `A,B,C`는 각각 인덱스 `0,1,2`에 대응합니다.
문자 코드를 이용해 필요한 값을 꺼내 순서대로 출력합니다.
