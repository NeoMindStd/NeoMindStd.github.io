# 풀이
## 문제

정수 `N` 이상이 되는 최소 정수 `x`에 대해
`x^2 >= N`을 만족하는 `x`를 구하는 문제입니다.

## 코드

```python
import math;print(math.ceil(int(input())**.5))
```

## 설명

`sqrt(N)`의 올림값이 조건을 만족하는 최소 정수 제곱근입니다.
