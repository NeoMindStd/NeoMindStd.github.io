# 풀이

## 문제

백준 2750번 문제를 풀이합니다.

## 코드

{% raw %}
`python
nums = [int(input()) for _ in range(int(input()))]

nums.sort()

d = [0]*8001
for n in nums:
    d[n+4000] += 1
tmp = max(d)
most = 4001
for i in range(-4000, 4001):
    if d[i+4000]==tmp:
        if most > i:
            most = i
        else:
            most = i
            break
        
print(round(sum(nums)/len(nums)))
print(nums[len(nums)//2])
print(most)
print(abs(max(nums)-min(nums)))

```
