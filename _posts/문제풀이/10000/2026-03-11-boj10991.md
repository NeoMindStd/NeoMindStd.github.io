# 풀이

## 문제

예제를 보고 규칙을 유추한 뒤에 별을 찍어 보세요.

**입력 요약**  
첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.

**출력 요약**  
첫째 줄부터 N번째 줄까지 차례대로 별을 출력한다.

## 코드

{% raw %}
```python
n = int(input())

for i in range(n):
    for j in range(n-i-1):
        print(' ', end='')
    print('*', end='')
    for j in range(2*i-1):
        print(' ' if j % 2 == 0 else '*', end='')
        
    if i != 0: print('*')
    else: print()
```
{% endraw %}
