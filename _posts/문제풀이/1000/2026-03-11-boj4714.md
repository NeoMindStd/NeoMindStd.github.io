# 풀이

## 문제

지구에서의 무게가 주어질 때
달에서의 무게(지구의 `0.167`배)를 출력하는 문제입니다.

## 코드

```python
while True:
    n = float(input())
    if n < 0: break
    print("Objects weighing %.2f on Earth will weigh %.2f on the moon." %(n, n*0.167))
```

## 설명

음수 입력이 나오기 전까지 반복하며
포맷 문자열에 맞춰 소수 둘째 자리까지 출력합니다.
