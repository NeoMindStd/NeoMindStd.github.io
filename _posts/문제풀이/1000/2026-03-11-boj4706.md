# 풀이

## 문제

속도 비율을 이용해 상대론적 식
`sqrt(1 - (tb^2/ta^2))` 값을 계산해 소수 셋째 자리까지 출력하는 문제입니다.

## 코드

```python
while True:
    ta,tb=map(float,input().split())
    if ta==tb==0:break
    print('%.3f'%((1-tb*tb/(ta*ta))**.5))
```

## 설명

입력 쌍마다 공식을 그대로 적용하고,
종료 조건 `0 0`까지 반복 출력합니다.
