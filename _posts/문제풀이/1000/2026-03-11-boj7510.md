# 풀이

## 문제

세 변 길이가 직각삼각형을 이루는지 판별해
시나리오별로 `yes/no`를 출력하는 문제입니다.

## 코드

```python
r=[]
for T in range(int(input())):
    l=sorted(map(int, input().split()))
    s=f'Scenario #{T+1}:'
    if l[2]**2==l[1]**2+l[0]**2:s+='\nyes'
    else:s+='\nno'
    r.append(s)
print('\n\n'.join(r))
```

## 설명

세 변을 정렬해 가장 긴 변을 빗변으로 놓고,
피타고라스 정리를 만족하는지 확인합니다.
