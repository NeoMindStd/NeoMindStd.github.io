# 무엇인가

Cheat-O-Matic은 오래된 세대의 메모리 검색 도구입니다. 현재는 공식 사이트가 사실상 종료돼 있어, 현역 유틸리티보다는 아카이브 대상에 가깝게 보는 편이 맞습니다.

# 공식 사이트와 다운로드 경로

- 공식 사이트: [Cheat-O-Matic](https://web.archive.org/web/20130529231036/http://cheat-o-matic.com/)
- 다운로드: [공식 다운로드 페이지](https://web.archive.org/web/20130529231036/http://cheat-o-matic.com/)
- 참고: 현재는 유지 중인 공식 배포처가 없어 Wayback에 남은 옛 공식 페이지 기준으로만 확인할 수 있습니다.

# 지원 OS와 설치 방법

## Windows

공식 다운로드 페이지에서 Windows용 설치 파일이나 패키지를 내려받아 설치합니다. 설치가 끝나면 첫 실행과 버전 확인을 한 번 진행하는 편이 좋습니다. 실제 설치가 필요하다면 주력 PC보다 격리된 테스트 환경이나 가상 머신에서만 다루는 편이 안전합니다.

# 기본 사용법

1. 과거 방식대로라면 대상 프로그램을 실행한 뒤 현재 값을 기준으로 첫 검색을 합니다.
2. 값이 바뀐 뒤 다시 검색해 후보를 줄이고, 결과 주소를 확인합니다.
3. 다만 현재는 실제 활용보다 레거시 참고 자료로 이해하는 편이 더 현실적입니다.

# 주의할 점

이 도구의 핵심 정보는 기능보다도 `공식 배포가 끝났다`는 점입니다. 현대 환경에서는 보안성과 호환성 모두 불확실하므로 현역 대안을 먼저 검토하는 편이 좋습니다.

# 요약

Cheat-O-Matic은 현재 실사용 권장 도구라기보다 역사적 참고 대상입니다. 공식 배포처가 유지되지 않는다는 사실 자체가 가장 중요한 포인트입니다.
