# 풀이
## 문제

2진수 문자열을 8진수 문자열로 변환해 출력하는 문제입니다.

## 코드

```python
print(format(int(input(),2),'o'))
```

## 설명

`int(input(), 2)`로 2진수 문자열을 정수로 변환하고,
`format(..., 'o')`로 8진수 문자열로 출력합니다.
