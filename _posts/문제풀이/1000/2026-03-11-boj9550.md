# 풀이

## 문제

상자별 사탕 개수와 1봉지당 필요한 사탕 수 `k`가 주어질 때
만들 수 있는 봉지 수를 구하는 문제입니다.

## 코드

```python
for T in range(int(input())):
    n,k=map(int,input().split())
    l=list(map(int,input().split()))
    print(sum(map(lambda x:x//k,l)))
```

## 설명

각 상자에서 만들 수 있는 봉지 수 `x//k`를 계산해
전부 합산하면 정답입니다.
