# 풀이
## 문제

여러 박스의 용량과 책 두께가 주어질 때,
책을 순서대로 담고 남는 박스 공간 총합을 구하는 문제입니다.

## 코드

```python
n, m = map(int,input().split())

boxes = list(map(int,input().split()))
books = list(map(int,input().split()))

i = j = 0
while i < n and j < m:
    if boxes[i] >= books[j]:
        boxes[i] -= books[j]
        j += 1
    else: i += 1
    
print(sum(boxes))
```

## 설명

현재 박스(`i`)에 현재 책(`j`)을 넣을 수 있으면 넣고,
못 넣으면 다음 박스로 넘어갑니다.

모든 책을 담은 뒤 박스 배열에 남은 용량 합을 출력합니다.
