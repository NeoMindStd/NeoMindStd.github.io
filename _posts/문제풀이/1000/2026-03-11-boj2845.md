# 풀이
## 문제

신문 기사에 나온 참가자 수 5개와
실제 사람 수 `L*P`가 주어질 때
각 기사 수치의 오차를 출력하는 문제입니다.

## 코드

```python
l, p = map(int, input().split())
print(*list(map(lambda x: x-l*p, list(map(int, input().split())))))
```

## 설명

기준값 `L*P`를 계산하고
각 기사 수치에서 빼서 오차를 출력합니다.
