# 풀이
## 문제

OX 결과에서
연속 정답이면 점수가 누적 증가하는 규칙으로
총점을 계산하는 문제입니다.

## 코드

```python
_,l=input(),input().split('0')
l=list(map(lambda x:x.split(), l))
for i in range(len(l)):
    t=sum(map(int,l[i]))
    l[i]=t*(t+1)//2
print(sum(l))
```

## 설명

`0` 기준으로 정답 구간을 분리한 뒤
각 구간 길이를 `t`라 하면
해당 구간 점수는 `1+...+t = t*(t+1)//2`입니다.
