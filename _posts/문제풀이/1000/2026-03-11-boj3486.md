# 풀이

## 문제

두 수를 각각 뒤집어 더한 뒤,
합을 다시 뒤집어 출력하는 문제입니다.

## 코드

```python
for T in range(int(input())):
    a, b = map(lambda x:int(x[::-1]), input().split())
    print(int(str(a+b)[::-1]))
```

## 설명

입력 문자열을 뒤집어 정수로 바꾸고 더한 뒤,
결과를 다시 뒤집어 정수로 출력하면 선행 0 처리까지 자동으로 해결됩니다.
