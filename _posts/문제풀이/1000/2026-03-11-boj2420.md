# 풀이
## 문제

두 수 `N`, `M`의 차이의 절댓값을 구하는 문제입니다.

## 코드

```python
n, m = map(int, input().split())
print(abs(n-m))
```

## 설명

문제 조건 그대로 `abs(N-M)`를 출력하면 됩니다.
