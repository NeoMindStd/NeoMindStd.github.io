# 풀이

## 문제

입금/출금 거래를 여러 줄 입력받아 잔액을 갱신하고,
잔액이 `-200` 미만이면 `Not allowed`를 출력하는 문제입니다.

## 코드

```python
while True:
    a,b,c=input().split()
    a,c=int(a),int(c)
    if a==c==0 and b=='W':break
    o='+'if b=='D'else'-'
    r=eval(f'{a}{o}{c}')
    if r<-200:print('Not allowed')
    else:print(r)
```

## 설명

거래 종류가 `D`면 입금, `W`면 출금으로 처리하고
계산 결과가 `-200`보다 작은지만 판별해 출력하면 됩니다.
