# 풀이
## 문제

좌표 `(x, y)`에 있는 점이 직사각형 경계(`0 ≤ x ≤ w`, `0 ≤ y ≤ h`)로
이동할 때 필요한 최소 거리를 구하는 문제입니다.

## 코드

```python
x, y, w, h = tuple(map(int, input().split()))
print(min(min(max(x, w) - min(x, w), x), min(max(y, h) - min(y, h), y)))
```

## 설명

경계까지의 후보 거리는 네 가지입니다.

 - 왼쪽: `x`
 - 오른쪽: `w - x`
 - 아래: `y`
 - 위: `h - y`

이 중 최소값을 출력하면 됩니다.
