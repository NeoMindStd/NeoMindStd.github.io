# 풀이
## 문제

거스름돈(센트)이 주어질 때
쿼터, 다임, 니켈, 페니의 최소 개수를 출력하는 문제입니다.

## 코드

```python
for T in range(int(input())):
    n=int(input())
    l=[25,10,5,1]
    r=[]
    for i in l:
        r.append(n//i)
        n%=i
    print(*r)
```

## 설명

큰 단위부터 나누는 그리디 방식으로
각 동전 개수를 순서대로 계산합니다.
