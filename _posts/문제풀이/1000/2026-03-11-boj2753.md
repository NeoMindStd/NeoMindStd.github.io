# 풀이
## 문제

입력 연도가 윤년이면 `1`,
아니면 `0`을 출력하는 문제입니다.

## 코드

```python
n = int(input())
print(1 if n % 400 == 0 or (n % 4 == 0 and n % 100 != 0) else 0)
```

## 설명

윤년 조건을 그대로 논리식으로 구현해
삼항 연산자로 출력합니다.
