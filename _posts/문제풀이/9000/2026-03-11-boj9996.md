# 풀이

## 문제

와일드카드 `*`가 포함된 패턴과 문자열들이 주어질 때 매칭 여부를 판별하는 문제입니다.

## 코드

```python
import re
n=int(input())
s=input()
r=''
for c in s:
    if c=='*':r+='.*';continue
    r+=c
p=re.compile(r)
for i in range(n):
    t=input()
    m=p.match(t)
    print('DA'if m is not None and m.group()==t else'NE')

```

## 설명

패턴의 `*`를 정규표현식 `.*`로 바꿔 컴파일한 뒤, 전체 문자열이 일치하는지 검사해 `DA/NE`를 출력합니다.
