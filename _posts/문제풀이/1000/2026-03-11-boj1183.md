# 풀이

## 문제

백준 1183번 문제를 풀이합니다.

## 코드

{% raw %}
`python
n=int(input())
d={}
for i in range(n):
    s,a=map(int,input().split())
    try:d[s-a]+=1
    except:d[s-a]=1

print(d)
print(list(d.items()))

```
{% endraw %}
