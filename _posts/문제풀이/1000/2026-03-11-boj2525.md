# 풀이
## 문제

현재 시각(시, 분)과 조리 시간(분)이 주어질 때
조리 완료 시각을 계산하는 문제입니다.

## 코드

```python
a, b = input().split()
a = int(a)
b = int(b)

c = int(input())

b += c

a += b//60

a %= 24

b %= 60

print(a, b)
```

## 설명

분에 조리 시간을 더한 뒤
`60`으로 나눈 몫을 시에 반영하고,
시는 `24`로, 분은 `60`으로 나머지 처리합니다.
