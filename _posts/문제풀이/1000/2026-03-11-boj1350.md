# 풀이
## 문제

파일 크기 목록과 클러스터 크기 `S`가 주어질 때,
각 파일이 차지하는 디스크 공간 총합을 구하는 문제입니다.

## 코드

```python
import math
input()
l=map(int,input().split())
s=int(input())
ea=0
for file in l: ea+=math.ceil(file/s)
print(ea*s)
```

## 설명

각 파일은 최소 1개 이상의 클러스터를 차지하므로
`ceil(file_size / S)`로 필요한 클러스터 수를 구합니다.

모든 파일의 클러스터 수를 합한 뒤
`S`를 곱해 실제 사용 공간을 계산합니다.
