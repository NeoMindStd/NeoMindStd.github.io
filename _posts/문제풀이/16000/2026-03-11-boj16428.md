# 풀이

## 문제

두 정수 A와 B를 입력받은 다음, A/B의 몫과 나머지를 출력하는 프로그램을 작성하시오.

어떤 정수 q와 r에 대해 A = qB + r (0 ≤ r < |B|)로 나타낼 수 있을 때, q를 몫, r을 나머지라고 한다.

**입력 요약**  
첫째 줄에 A와 B가 주어진다. (-1010000 ≤ A, B ≤ 1010000, B ≠ 0)

**출력 요약**  
첫째 줄에 A/B의 몫, 둘째 줄에 나머지를 출력한다.

## 코드

{% raw %}
```python
a,b=map(int,input().split())
t=1 if b>0 else -1
print(a//(t*b)*t,a%(t*b),sep='\n')
```
{% endraw %}

## 설명

핵심은 `구현` 관점에서 두 정수 A와 B를 입력받은 다음, A/B의 몫과 나머지를 출력하는 프로그램을 작성하시오.

어떤 정수 q와 r에 대해 A = qB + r (0 ≤ r < |B|)로 나타낼 수 있을 때, q를 몫, r을 나머지라고 한다.를 만족하도록 로직을 구성하는 것입니다.

코드는 입력을 파싱한 뒤 조건 분기와 계산을 순서대로 수행하고, 문제에서 요구한 형식으로 결과를 출력합니다.
