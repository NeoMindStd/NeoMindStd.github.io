# 풀이

## 문제

등차수열에서 세 수가 주어졌을 때
비어 있는 네 번째 수를 찾는 문제입니다.

## 코드

```python
l=sorted(map(int,input().split()))
a,b=l[1]-l[0],l[2]-l[1]
if a>b:print((l[1]+l[0])//2)
elif a<b:print((l[2]+l[1])//2)
else:print(l[2]+a)
```

## 설명

정렬 후 인접한 차이를 비교하면
빠진 값이 앞 구간인지 뒤 구간인지, 혹은 끝에 붙는지 판별할 수 있습니다.
