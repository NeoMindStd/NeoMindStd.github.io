# 풀이

## 문제

주어진 N개의 정수 중에서 양의 정수의 개수를 출력하는 프로그램을 작성하시오.

**입력 요약**  
첫째 줄에 최대 1,000,000개의 정수가 주어진다. 입력으로 주어지는 정수는 -1,000,000보다 크거나 같고, 1,000,000보다 작거나 같다.

**출력 요약**  
첫째 줄에 양의 정수의 개수를 출력한다.

## 코드

{% raw %}
```python
print(len(list(filter(lambda x:x>0,map(int,input().split())))))
```
{% endraw %}

## 설명

핵심은 `구현` 관점에서 주어진 N개의 정수 중에서 양의 정수의 개수를 출력하는 프로그램을 작성하시오.를 만족하도록 로직을 구성하는 것입니다.

코드는 입력을 파싱한 뒤 조건 분기와 계산을 순서대로 수행하고, 문제에서 요구한 형식으로 결과를 출력합니다.
