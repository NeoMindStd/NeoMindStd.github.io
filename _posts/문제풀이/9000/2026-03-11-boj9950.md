# 풀이

## 문제

직사각형의 세 값 중 하나가 0으로 주어질 때 나머지 값을 계산해 출력하는 문제입니다.

## 코드

```python
import sys;read=sys.stdin.readline
r=''
while True:
    l,w,a=map(int,read().split())
    if l==w==a==0:break
    elif l==0:l=a//w
    elif w==0:w=a//l
    else:a=w*l
    r+=f'{l} {w} {a}\n'
print(r)

```

## 설명

입력을 반복 처리하면서 0인 항목을 나머지 두 값으로 계산하고, 종료 조건(0 0 0)에서 반복을 멈춥니다.
