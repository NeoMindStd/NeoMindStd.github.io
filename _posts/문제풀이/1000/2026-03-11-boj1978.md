# 풀이
## 문제

입력된 수들 중 소수의 개수를 세는 문제입니다.

## 코드

```python
a = int(input())
datas = input().split()
datas = list(datas)

count = 0
for i in datas:
    i = int(i)
    flag = True
    for j in range(2,i//2+1):
        if(i%j == 0):
            flag = False
            break
    if(flag and i > 1):
        count+=1
print(count)
```

## 설명

각 수에 대해 `2`부터 `i//2`까지 나눠보며
나누어떨어지는 수가 있으면 소수가 아니라고 판정합니다.

`1`을 제외한 소수 개수를 최종 출력합니다.
