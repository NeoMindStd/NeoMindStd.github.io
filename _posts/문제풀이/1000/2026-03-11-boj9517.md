# 풀이

## 문제

폭탄 타이머(210초) 동안 질문/정답 기록이 주어질 때
폭탄이 터질 시점의 플레이어 번호를 구하는 문제입니다.

## 코드

```python
k,n=int(input())-1,int(input())
l=[input().split() for _ in range(n)]
l=list(map(lambda x: (int(x[0]),x[1]),l))
s=0
for i in range(n):
    s+=l[i][0]
    if s>=210:break
    if l[i][1]=='T':k=(k+1)%8
print(k+1)
```

## 설명

시간을 누적하다가 `210` 이상이 되는 순간 중단하고,
그 전까지 정답(`T`)이면 다음 플레이어로 넘기며 최종 번호를 출력합니다.
