# 풀이
## 문제

현재 시각(시, 분, 초)과 추가할 초가 주어질 때
결과 시각을 계산하는 문제입니다.

## 코드

```python
t = list(map(int, input().split()))
time = t[0]*3600+t[1]*60+t[2]+int(input())
time%=24*3600
t[0]=time//3600
time%=3600
t[1]=time//60
t[2]=time%60
print(*t)
```

## 설명

시각을 전체 초로 변환해 더한 뒤
하루(`24*3600`) 기준 나머지 처리하고,
다시 시/분/초로 역변환하면 됩니다.
