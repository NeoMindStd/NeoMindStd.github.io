# 풀이

## 문제

숫자의 각 자릿수를 반복 합산해
한 자리 수(디지털 루트)가 될 때까지 계산하는 문제입니다.

## 코드

```python
while True:
    n=list(input())
    if n[0]=='0':break
    while len(n)>1:n=list(str(sum(map(int,n))))
    print(sum(map(int,n)))
```

## 설명

문자열 입력을 자릿수 합으로 줄여나가며
길이가 `1`이 되면 해당 값을 출력합니다.
