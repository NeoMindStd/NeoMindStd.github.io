# 풀이

## 문제
두 정수 A, B가 주어질 때, A/B를 출력

## 코드

```python
a, b = input().split()
a = int(a)
b = int(b)

print(a/b)
```

## 설명
a, b를 입력받아 나눗셈 연산결과를 출력합니다.
