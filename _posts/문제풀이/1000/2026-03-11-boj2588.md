# 풀이
## 문제

세 자리 수 곱셈 과정을
문제 형식에 맞춰 단계별로 출력하는 문제입니다.

## 코드

```python
a = int(input())
b = input()

print(a*int(b[2]))
print(a*int(b[1]))
print(a*int(b[0]))
print(a*int(b))
```

## 설명

문자열로 받은 두 번째 수의 각 자리 숫자를 사용해
일의 자리부터 곱한 결과를 출력하고,
마지막에 전체 곱을 출력합니다.
