# 무엇인가

Cakewalk는 Windows 오디오 제작 환경에서 꾸준히 언급되는 DAW 계열 도구입니다. 현재 공식 사이트 기준으로 제품 접근 방식과 계정 정책이 예전과 다를 수 있으므로, 설치 전에 현재 제공되는 제품 구성을 먼저 확인하는 편이 중요합니다.

# 공식 사이트와 다운로드 경로

- 공식 홈페이지 & 다운로드: [Cakewalk](https://www.cakewalk.com/)

# 지원 OS와 설치 방법

## Windows

공식 다운로드 페이지에서 Windows용 설치 파일이나 패키지를 내려받아 설치합니다. 설치가 끝나면 첫 실행과 버전 확인을 한 번 진행하는 편이 좋습니다. 오디오 인터페이스 드라이버, ASIO 설정, 플러그인 경로를 처음에 정리해 두면 이후 작업이 편합니다.

## macOS

현재 공식 macOS 배포를 확인하기 어렵습니다. 공식 macOS 데스크톱 지원은 일반적으로 제공되지 않습니다.

## Linux

현재 공식 Linux 배포를 확인하기 어렵습니다. 공식 Linux 데스크톱 지원은 제공되지 않습니다.

# 기본 사용법

1. 새 프로젝트를 만들고 오디오 인터페이스, 샘플레이트, 버퍼 사이즈를 먼저 확인합니다.
2. 오디오 트랙이나 MIDI 트랙을 추가하고 입력 장치를 지정한 뒤 짧게 녹음합니다.
3. 믹서와 이펙트 체인을 조정하고 마지막에 오디오 파일로 익스포트합니다.

# 주의할 점

DAW는 설치보다 오디오 장치와 드라이버 설정이 더 중요합니다. 지연시간이 크거나 입력이 안 잡히는 문제는 프로그램보다 인터페이스 설정에서 오는 경우가 많습니다.

# 요약

Cakewalk는 Windows 오디오 제작을 시작할 때 여전히 의미 있는 이름입니다. 공식 사이트에서 현재 제품 구조를 먼저 확인하고, 오디오 장치 세팅부터 차분히 맞추는 것이 중요합니다.
