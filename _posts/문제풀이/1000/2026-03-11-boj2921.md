# 풀이

## 문제

`0`부터 `N`까지의 숫자 쌍 `(i, j)` (`i <= j`)의 합을 모두 더하는 문제입니다.

## 코드

```python
n=int(input())
print(n*(n+1)*(n+2)//2)
```

## 설명

합을 전개하면 닫힌 형태인 `N(N+1)(N+2)/2`로 정리되므로 바로 계산할 수 있습니다.
