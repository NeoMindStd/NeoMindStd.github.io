# 풀이
## 문제

정수 `N`이 주어질 때
`N`부터 `1`까지 한 줄에 하나씩 출력하는 문제입니다.

## 코드

```python
a = int(input())

for i in range(a, 0, -1):
    print(i)
```

## 설명

감소 범위 반복문을 사용해
`N`에서 `1`까지 역순 출력합니다.
