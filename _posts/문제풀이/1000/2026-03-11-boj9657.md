# 풀이

## 문제

돌 `n`개에서 1, 3, 4개를 가져갈 수 있을 때
최적 플레이 기준 승자를 구하는 문제입니다.

## 코드

```python
n=int(input())
l=[False]*(n+1)
for i in range(n):
    try:
        if not l[i]:
            l[i+1]=True
            l[i+3]=True
            l[i+4]=True
    except:pass
print('SK'if l[-1]else'CY')
```

## 설명

현재 상태가 패배 상태면 다음(1/3/4개 이동) 상태를 승리로 표시하는 방식으로
도달 가능 DP를 채워 최종 승자를 판단합니다.
