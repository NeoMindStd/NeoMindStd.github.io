# 풀이

## 문제

책 총 가격과 9권 가격이 주어졌을 때,
빠진 1권 가격을 구하는 문제입니다.

## 코드

```python
price = int(input())
for _ in range(9):
    price -= int(input())
print(price)
```

## 설명

총액에서 입력되는 9권 가격을 차례로 빼면
남는 값이 빠진 책의 가격입니다.
