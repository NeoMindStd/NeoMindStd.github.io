# 풀이

## 문제

각 테스트케이스에서 두 주사위 눈의 합을
`Case i:` 형식으로 출력하는 문제입니다.

## 코드

```python
for T in range(int(input())):print('Case %d: %d'%(T+1,sum(map(int,input().split()))))
```

## 설명

입력 두 수를 더해
케이스 번호와 함께 형식에 맞춰 출력합니다.
