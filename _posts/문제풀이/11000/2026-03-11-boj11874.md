# 풀이

## 문제

백준 11874번 문제를 풀이합니다.

## 코드

{% raw %}
`python
l,d,x=(int(input())for _ in range(3))
n=m=0
for i in range(d-l+1):
    if n==0 and sum(map(int,str(l+i)))==x:n=l+i
    if m==0 and sum(map(int,str(d-i)))==x:m=d-i
    if n!=0!=m:break
print(n,m,sep='\n')

```
{% endraw %}
