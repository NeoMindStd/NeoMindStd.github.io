# 풀이
## 문제

주어진 수를
일의 자리부터 차례로 반올림해 최종 값을 구하는 문제입니다.

## 코드

```python
n = list(map(int, input()))

for i in range(len(n)-1, 0, -1):
    if n[i] > 4:
        n[i-1] += 1
    n[i] = 0

print(''.join(map(str,n)))
```

## 설명

오른쪽 자리부터 보면서
현재 자리가 5 이상이면 바로 왼쪽 자리 올림,
현재 자리는 0으로 만듭니다.

이를 끝까지 반복해 최종 반올림 결과를 출력합니다.
