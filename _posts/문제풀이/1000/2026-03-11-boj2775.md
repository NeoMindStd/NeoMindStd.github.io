# 풀이
## 문제

아파트 `k`층 `n`호의 거주 인원을 규칙에 따라 계산하는 문제입니다.

## 코드

```python
n = int(input())

fams = [[i for i in range(1,15)]]
for i in range(15):
    tmp = [1]
    for j in range(13):
        tmp.append(tmp[j]+fams[i][j+1])
    fams.append(tmp)

for i in range(n):
    a, b = int(input()), int(input())
    print(fams[a][b-1])
```

## 설명

0층부터 규칙대로 인원을 미리 DP 테이블로 계산해 두고,
질의마다 해당 층/호 값을 바로 출력합니다.
