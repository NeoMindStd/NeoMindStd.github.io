# 풀이

## 문제

트리블 수열의 `n`번째 값을
여러 테스트케이스에 대해 출력하는 문제입니다.

## 코드

```python
koong = [1, 1, 2, 4]

for i in range(4, 69):
    koong.append(koong[i-1]+koong[i-2]+koong[i-3]+koong[i-4])

for _ in range(int(input())):
    print(koong[int(input())])
```

## 설명

점화식 `K[n]=K[n-1]+K[n-2]+K[n-3]+K[n-4]`를
최대 입력 범위까지 미리 계산해 질의에 답합니다.
