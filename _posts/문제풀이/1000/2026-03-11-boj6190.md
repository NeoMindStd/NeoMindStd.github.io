# 풀이

## 문제

주어진 수에 대해 규칙을 반복 적용해
`1`이 될 때까지의 단계 수를 구하는 문제입니다.

## 코드

```python
import sys;read=sys.stdin.readline
n,s=int(input()),0
while True:
    if n==1:break
    if n%2==0:n//=2
    else:n=n*3+1
    s+=1
print(s)
```

## 설명

짝수면 `n/2`, 홀수면 `3n+1`을 반복하면서
`n == 1`이 될 때까지 연산 횟수를 누적합니다.
