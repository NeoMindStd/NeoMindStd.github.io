# 풀이
## 문제

정수 집합 `S`가 주어질 때,
숫자 `n`을 포함하면서 `S`의 어떤 원소도 포함하지 않는 구간 `[A, B]`의 개수를 구하는 문제입니다.

단, `A < n < B`이고 `A`, `B`는 자연수입니다.

## 코드

```python
l=int(input())
s=[0]+list(sorted(map(int,input().split())))
n=int(input())
i=0
while s[i]<n:i+=1
a=(n-s[i-1])*(s[i]-n)-1
print(a if a>0 else 0)
```

## 설명

`n` 바로 왼쪽 경계값을 `L`, 오른쪽 경계값을 `R`이라 하면
가능한 구간 수는

`(n - L) * (R - n) - 1`

로 계산됩니다.

코드는 정렬된 배열에서 `n`이 들어갈 위치를 찾아
위 공식을 적용하고, 음수가 나오는 경우 `0`을 출력합니다.
