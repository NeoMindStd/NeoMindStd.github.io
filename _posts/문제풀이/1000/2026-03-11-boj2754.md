# 풀이
## 문제

학점 문자열(`A+`, `B0`, `F` 등)이 주어질 때
해당 평점 값을 출력하는 문제입니다.

## 코드

```python
print({'A+' : 4.3, 'A0': 4.0, 'A-': 3.7,\
'B+': 3.3, 'B0': 3.0, 'B-': 2.7,\
'C+': 2.3, 'C0': 2.0, 'C-': 1.7,\
'D+': 1.3, 'D0': 1.0, 'D-': 0.7,\
'F': 0.0}[input()])
```

## 설명

딕셔너리 매핑으로
입력 문자열을 즉시 대응 점수로 변환해 출력합니다.
