# 풀이
## 문제

나무 성장 단계별로
곱셈/뺄셈 연산을 순서대로 적용해
최종 생장점 개수를 구하는 문제입니다.

## 코드

```python
while True:
    n=list(map(int,input().split()))
    if n[0]==0:break
    s=1
    for i in range(1,n[0]*2,2):
        s*=n[i]
        s-=n[i+1]
    print(s)
```

## 설명

각 테스트 케이스마다 초기값 `s=1`에서 시작해
`(곱하기, 빼기)` 쌍을 순서대로 적용합니다.

입력이 `0`으로 시작하면 종료합니다.
