# 풀이

## 문제

세 변 길이가 주어질 때
삼각형 종류(정/이등변/부등변/불가능)를 판별하는 문제입니다.

## 코드

```python
while True:
    l=sorted(map(int,input().split()))
    if l[2]==0:break
    if l[0]==l[2]:print('Equilateral')
    elif l[0]+l[1]<=l[2]:print('Invalid')
    elif l[0]==l[1]or l[1]==l[2]:print('Isosceles')
    else:print('Scalene')
```

## 설명

정렬 후 가장 긴 변을 기준으로 삼각형 성립 여부를 먼저 확인하고,
변의 일치 개수에 따라 종류를 판정합니다.
