# 풀이
## 문제

한 줄의 문자열이 주어질 때
문자열 안에 포함된 단어의 개수를 구하는 문제입니다.

## 코드

```python
print(len(input().split()))
```

## 설명

파이썬의 `split()`은 공백 기준으로 단어를 분리하고,
앞뒤 연속 공백을 자동으로 처리합니다.

따라서 분리된 리스트 길이를 출력하면 단어 수가 됩니다.
