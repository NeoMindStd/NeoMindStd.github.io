# 풀이

## 문제

현재 점수 `k`에서 목표 점수 `w` 이상이 되기 위해
한 번에 `m`점씩 올릴 때 필요한 횟수를 구하는 문제입니다.

## 코드

```python
import math
k,w,m=map(int,input().split())
print(math.ceil((w-k)/m))
```

## 설명

부족한 점수 `w-k`를 `m`으로 나눈 뒤
올림 처리하면 최소 횟수가 됩니다.
