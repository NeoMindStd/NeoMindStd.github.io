# 풀이

## 문제

조건을 만족하는 수의 개수를 구하는 문제로,
코드는 상한을 이용해 경우의 수를 세는 방식으로 작성되어 있습니다.

## 코드

```python
n=int(input())
r=0
for i in range(min(100,n)):
    if n-i<100:r+=1
print(r)
```

## 설명

`i`를 0부터 순회하면서 조건 `n-i < 100`을 만족하는 개수를 카운트합니다.
반복 범위를 `min(100, n)`으로 제한해 불필요한 탐색을 줄입니다.
