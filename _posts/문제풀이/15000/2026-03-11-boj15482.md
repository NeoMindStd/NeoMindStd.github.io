# 풀이

## 문제

백준 15482번 문제를 풀이합니다.

## 코드

{% raw %}
`python
s1, s2 = input(), input()
dp = [[0]*(len(s2)+1) for _ in range(len(s1)+1)]
for i in range(1, len(s1)+1):
    for j in range(1, len(s2)+1):
        if s1[i-1] == s2[j-1]:
            dp[i][j] = dp[i-1][j-1]+1
        else:
            dp[i][j] = max(dp[i-1][j], dp[i][j-1])
m = 0
for row in dp:
    m = max(m, max(row))
print(m)

```
