# 풀이

## 문제

버거 3개와 음료 2개 가격이 주어질 때,
가장 저렴한 세트 메뉴 가격을 구하는 문제입니다.

## 코드

```python
print(min([int(input()) for _ in range(3)])+min([int(input()) for _ in range(2)])-50)
```

## 설명

버거 최솟값과 음료 최솟값을 더한 뒤,
세트 할인 50원을 빼서 출력합니다.
