# 풀이
## 문제

8개의 음이 주어질 때
오름차순이면 `ascending`, 내림차순이면 `descending`,
아니면 `mixed`를 출력하는 문제입니다.

## 코드

```python
n = list(map(int, input().split()))

mode = "mixed"
aFlag=True
bFlag=True
for i, ax in enumerate(n):
    if(i+1!=ax) : aFlag=False
    if(8-i!=ax) : bFlag=False
if(aFlag) : mode="ascending"
elif(bFlag) : mode="descending"
print(mode)
```

## 설명

입력 배열이 `1..8` 순서인지, `8..1` 순서인지 각각 검사해
결과 문자열을 결정합니다.
