# 풀이

## 문제

정상적인 체스 기물 개수와 현재 입력 개수의 차이를
순서대로 출력하는 문제입니다.

## 코드

```python
c = [1, 1, 2, 2, 2, 8]
n = list(map(int, input().split()))
for i in range(6):
    c[i]-=n[i]
print(*c)
```

## 설명

정답 기물 배열에서 입력 배열을 원소별로 빼면
추가/제거해야 하는 개수를 바로 얻을 수 있습니다.
