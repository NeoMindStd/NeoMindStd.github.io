# 풀이
## 문제

입력 `N`에 대해
앞 공백이 늘고 별은 줄어드는 오른쪽 정렬 패턴을 출력하는 문제입니다.

## 코드

```python
a = int(input())

for i in range(a):
    for j in range(i):
        print(" ", end="")
    for j in range(a-i):
        print("*", end="")
    print()
```

## 설명

각 줄마다 공백 `i`개, 별 `N-i`개를 출력해
문제의 모양을 만듭니다.
