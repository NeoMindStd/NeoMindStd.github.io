# 풀이

## 문제

전체 페이지 수 `n`과 찢어진 페이지 번호 `p`가 주어질 때,
같은 장(sheet)에 있던 나머지 3페이지를 찾는 문제입니다.

## 코드

```python
while True:
    n=input()
    if n=='0':break
    n,p=map(int,n.split())
    t=p+(1 if p%2!=0 else-1)
    print(*sorted((t,n-p+1,n-t+1)))
```

## 설명

같은 면의 짝 페이지를 먼저 구하고,
책의 앞뒤 대응 관계 `x + y = n + 1`을 이용해 나머지 페이지를 계산합니다.
