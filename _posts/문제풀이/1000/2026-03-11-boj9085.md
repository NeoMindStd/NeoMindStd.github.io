# 풀이

## 문제

각 테스트케이스마다 주어진 정수들의 합을
출력하는 문제입니다.

## 코드

```python
for T in range(int(input())):
    input();print(sum(map(int,input().split())))
```

## 설명

각 테스트에서 개수 입력은 소비만 하고,
다음 줄 정수들을 `sum`으로 더해 바로 출력합니다.
