# 풀이

## 문제

자연수 A, B가 주어지면 A+B를 구하는 프로그램을 작성하시오.

**입력 요약**  
자연수 A, B (0 < A, B ≤ 10)가 첫 번째 줄에 주어진다. 단, 두 수의 사이에는 공백이 주어지지 않는다. 두 수의 앞에 불필요한 0이 붙는 경우는 없다.

**출력 요약**  
첫 번째 줄에 A+B의 값을 출력한다.

## 코드

{% raw %}
```python
s = input()
flag = s[1] != '0'
a = s[0:1 if flag else 2]
b = s[len(a) : len(s)]
print(int(a)+int(b))
```
{% endraw %}

## 설명

핵심은 `구현` 관점에서 자연수 A, B가 주어지면 A+B를 구하는 프로그램을 작성하시오.를 만족하도록 로직을 구성하는 것입니다.

코드는 입력을 파싱한 뒤 조건 분기와 계산을 순서대로 수행하고, 문제에서 요구한 형식으로 결과를 출력합니다.
