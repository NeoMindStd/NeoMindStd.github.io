# 풀이

## 문제

각 테스트케이스마다 두 수의 최소공배수(LCM)를 구하는 문제입니다.

## 코드

```python
import sys;read=sys.stdin.readline
def gcd(a,b):
    while b>0:a,b=b,a%b
    return a
for T in range(int(read())):
    a,b=map(int,read().split())
    print(a*b//gcd(a,b))
```

## 설명

유클리드 호제법으로 최대공약수 `gcd`를 구하고
`LCM = a*b/gcd` 공식을 적용합니다.
