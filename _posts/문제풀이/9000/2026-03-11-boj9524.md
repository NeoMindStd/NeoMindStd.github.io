# 풀이

## 문제

입력으로 주어진 정수 인덱스에 따라 미리 정의된 값을 출력하는 문제입니다.

## 코드

```python
print([0,1,7,2,3][int(input())])

```

## 설명

문제에서 주어진 대응 값을 리스트로 두고, 입력값을 인덱스로 사용해 바로 출력합니다.
