# 풀이

## 문제

백준 10829번 문제를 풀이합니다.

## 코드

{% raw %}
`python
n = int(input())

bi = list()
bi.append(n%2)

while n > 0:
    n//=2
    bi.append(n%2)

bi=list(map(str, bi))
print(int(''.join(bi)[::-1]))

```
{% endraw %}
