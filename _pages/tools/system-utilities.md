# 시스템 점검·유틸리티 허브

PC 상태를 확인하거나, Windows 중심 유틸리티를 찾을 때 자주 쓰는 도구만 모았습니다.

### 작업 목적별 빠른 선택표

| 목적 | 먼저 볼 글 | 잘 맞는 상황 | 다음 단계 |
|:--|:--|:--|:--|
| CPU, 메모리, GPU 정보를 빠르게 보고 싶을 때 | [CPU-Z 설치와 사용법](/tools/cpu-z-guide/) | 시스템 사양과 센서를 바로 확인하고 싶을 때 | GPU-Z, CrystalDiskInfo |
| SSD/HDD 건강 상태를 확인하고 싶을 때 | [CrystalDiskInfo 설치와 사용법](/tools/crystaldiskinfo-guide/) | 저장장치 상태를 점검하고 싶을 때 | TestMem5 |
| 메모리 안정성을 테스트하고 싶을 때 | [TestMem5 설치와 사용법](/tools/testmem5-guide/) | 램 오버나 불안정 증상을 점검할 때 | CPU-Z, GPU-Z |
| 네트워크/Windows 유틸리티가 필요할 때 | [GoodbyeDPI 설치와 사용법](/tools/gui-for-dpi-guide/) | DPI 우회 같은 네트워크 유틸이 필요할 때 | Hitomi Downloader |
| 메모리 스캔류 도구를 참고하고 싶을 때 | [Cheat Engine 설치와 사용법](/tools/cheat-engine-guide/) | 교육용/기술적 이해 차원에서 메모리 스캔 흐름을 볼 때 | Cheat-O-Matic |

### 추천 순서

1. 가장 실용적인 시작점은 `CrystalDiskInfo`, `CPU-Z`, `GPU-Z`처럼 상태를 안전하게 확인하는 도구들입니다.
2. 안정성 점검이 목적이면 `TestMem5`까지 같이 보면 하드웨어 상태를 더 입체적으로 볼 수 있습니다.
3. `GoodbyeDPI`, `Hitomi Downloader`처럼 목적형 유틸리티는 필요한 상황이 분명할 때 보는 편이 좋습니다.
4. `Cheat Engine`, `Cheat-O-Matic`은 일반 유틸보다는 메모리 구조를 이해하는 쪽에 더 가깝다는 점을 염두에 두는 편이 좋습니다.
