# 풀이
## 문제

재산 `N`과 사람 수 `M`이 주어질 때
한 사람당 나눠 갖는 몫과 나머지를 출력하는 문제입니다.

## 코드

```python
a,b=map(int,input().split());print(a//b,a%b,sep='\n')
```

## 설명

정수 나눗셈 몫은 `//`,
나머지는 `%` 연산으로 계산합니다.

문제 형식에 맞게 몫과 나머지를 줄바꿈으로 출력합니다.
