# 무엇인가

GoldWave는 오래전부터 꾸준히 쓰이는 오디오 편집 도구입니다. 복잡한 DAW보다 빠르게 파일을 열고 잘라내거나 볼륨을 맞추고 간단한 효과를 적용하는 데 특히 편합니다.

# 공식 사이트와 다운로드 경로

- 공식 사이트: [GoldWave](https://goldwave.com/)
- 다운로드: [공식 다운로드 페이지](https://goldwave.com/download.php)

# 지원 OS와 설치 방법

## Windows

공식 다운로드 페이지에서 Windows용 설치 파일이나 패키지를 내려받아 설치합니다. 설치가 끝나면 첫 실행과 버전 확인을 한 번 진행하는 편이 좋습니다. 자주 쓰는 코덱이나 입출력 장치가 있다면 첫 실행 후 옵션에서 기본 장치를 맞춰 두는 편이 좋습니다.

# 기본 사용법

1. 오디오 파일을 열고 자르기, 복사, 페이드 같은 기본 편집을 먼저 수행합니다.
2. 볼륨 조절, 노이즈 감소, 정규화 같은 처리로 파일 상태를 다듬습니다.
3. 출력 형식과 비트레이트를 정한 뒤 새 파일로 저장해 원본을 보존합니다.

# 주의할 점

오디오 편집은 한 번 덮어쓰면 되돌리기 어려운 경우가 많습니다. 원본 파일을 별도로 보관하고, 효과 적용 전후를 비교 청취하는 습관이 중요합니다.

# 요약

GoldWave는 빠른 오디오 편집에 강한 Windows 중심 도구입니다. 복잡한 믹싱보다 파일 단위 편집과 정리 작업에 특히 잘 맞습니다.
