# 풀이

## 문제

합이 `n`이 되는 두 주사위 눈 조합을
조건에 맞게 출력하는 문제입니다.

## 코드

```python
for i in range(int(input())):
    n=int(input())
    print(f'Case {i+1}:')
    s=[]
    for i in range(max(1,n-6),min(n//2,12)+1):s.append(f'({i},{n-i})')
    print(*s,sep='\n')
```

## 설명

첫 눈 `i` 범위를 유효 구간으로 제한하고
중복 없는 `(i, n-i)` 쌍을 출력합니다.
