# 풀이

## 문제

첫 번째 링과 나머지 링의 회전 수 비율을
기약분수 형태로 출력하는 문제입니다.

## 코드

```python
def gcd(a, b):
    while b != 0:
        a, b = b, a % b
    
    return a

input()
rings = list(map(int, input().split()))

for i in range(1, len(rings)):
    g = gcd(rings[0], rings[i])
    print("%d/%d" %(rings[0]//g, rings[i]//g))
```

## 설명

비율 `rings[0] / rings[i]`를 최대공약수로 나눠
기약분수로 출력하면 됩니다.
