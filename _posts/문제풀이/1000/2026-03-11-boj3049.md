# 풀이

## 문제

정점 `n`개인 볼록 다각형에서
두 대각선이 만나는 교점의 개수를 구하는 문제입니다.

## 코드

```python
n=int(input())
print(n*(n-1)*(n-2)*(n-3)//24)
```

## 설명

교점 하나는 서로 다른 4개 정점을 선택할 때마다 정확히 하나씩 생기므로
정답은 조합식 `nC4`입니다.
