# 풀이

## 문제

민국이와 만세의 4과목 점수를 각각 더한 뒤,
더 큰 총점을 출력하는 문제입니다.

## 코드

```python
print(max(sum(map(int,input().split())),sum(map(int,input().split()))))
```

## 설명

두 줄 입력을 각각 합산해
`max`로 큰 값을 바로 출력합니다.
