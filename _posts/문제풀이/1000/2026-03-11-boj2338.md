# 풀이
## 문제

매우 큰 정수 두 개에 대해
덧셈, 뺄셈, 곱셈 결과를 출력하는 문제입니다.

## 코드

```python
a,b=int(input()),int(input());print(a+b,a-b,a*b,sep='\n')
```

## 설명

파이썬의 arbitrary precision 정수를 그대로 활용해
세 연산 결과를 줄바꿈으로 출력합니다.
