# 풀이

## 문제

입력 전체에서 단어(`a-z`, `-`)만 남기고
가장 긴 단어를 찾아 출력하는 문제입니다.

## 코드

```python
s = ''

while True:
    s += input()
    if s[-5:]=='E-N-D':
        s = s[:-5]
        break

s = ''.join(map(lambda x: x if ord('a')<=ord(x)<=ord('z') or x=='-' else ' ', s.lower()))
print(max(s.split(), key=lambda x:len(x)))
```

## 설명

`E-N-D` 전까지 문자열을 누적한 뒤,
유효 문자를 제외한 나머지를 공백 처리해서 최장 단어를 찾습니다.
