# 풀이

## 문제

비트열 마지막 문자가 `e` 또는 `o`일 때
짝수/홀수 패리티가 되도록 마지막 비트를 채우는 문제입니다.

## 코드

```python
while True:
    s=input()
    if s=='#':break
    r=0
    for c in s:
        if c=='1':r+=1
    r='0'if r%2==0 and s[-1]=='e' or r%2!=0 and s[-1]=='o' else '1'
    print(s[:-1]+r)
```

## 설명

`1`의 개수를 세고 목표 패리티(`e`/`o`)에 맞춰
마지막 비트를 `0` 또는 `1`로 결정해 붙여 출력합니다.
