# 풀이
## 문제

다섯 정수가 주어질 때
평균과 중앙값을 출력하는 문제입니다.

## 코드

```python
l=[int(input())for _ in range(5)]
print(sum(l)//5)
print(sorted(l)[2])
```

## 설명

평균은 합을 `5`로 나눈 정수값,
중앙값은 정렬 후 가운데(인덱스 `2`) 값입니다.
