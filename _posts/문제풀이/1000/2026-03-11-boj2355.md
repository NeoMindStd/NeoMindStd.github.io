# 풀이
## 문제

두 정수 `A`, `B` 사이의 모든 정수 합을 구하는 문제입니다.

(`A`가 `B`보다 클 수도 있음)

## 코드

```python
l=sorted(list(map(int,input().split())))
n=l[1]-l[0]+1
print(n*l[0]+n*(n-1)//2)
```

## 설명

구간을 오름차순으로 맞춘 뒤
등차수열 합 공식
`n * first + n*(n-1)/2`
를 사용해 합을 계산합니다.
