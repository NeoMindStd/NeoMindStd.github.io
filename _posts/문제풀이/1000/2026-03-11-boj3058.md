# 풀이

## 문제

각 테스트케이스마다 7개 수 중 짝수만 골라
합과 최솟값을 출력하는 문제입니다.

## 코드

```python
for T in range(int(input())):
    l=list(filter(lambda x:x%2==0,map(int,input().split())))
    print(sum(l),min(l))
```

## 설명

짝수만 필터링한 리스트를 만든 뒤
`sum`, `min`을 출력하면 됩니다.
