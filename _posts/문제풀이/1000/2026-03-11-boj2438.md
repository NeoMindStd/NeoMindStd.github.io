# 풀이
## 문제

입력 `N`에 대해
첫 줄부터 `1`개, `2`개 ... `N`개의 별을 출력하는 문제입니다.

## 코드

```python
a = int(input())

for i in range(a):
    for j in range(i+1):
        print("*", end="")
    print()
```

## 설명

바깥 루프로 줄 수를,
안쪽 루프로 해당 줄의 별 개수를 제어해 출력합니다.
