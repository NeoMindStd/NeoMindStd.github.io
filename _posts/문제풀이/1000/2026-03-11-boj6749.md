# 풀이

## 문제

두 형제 나이가 주어졌을 때
등차 관계를 이용해 막내의 나이를 구하는 문제입니다.

## 코드

```python
a, b = int(input()), int(input())
print(b-a+b)
```

## 설명

두 입력의 차이를 한 번 더 더해
`2*b - a`를 출력하면 됩니다.
