# 풀이
## 문제

9개의 수가 주어질 때
최댓값과 그 값의 위치(1-based)를 출력하는 문제입니다.

## 코드

```python
nums = list()
for _ in range(9):
    nums.append(int(input()))
print(max(nums))
print(nums.index(max(nums))+1)
```

## 설명

리스트에 입력을 저장한 뒤
`max`로 최댓값을 구하고,
`index` 결과에 `1`을 더해 위치를 출력합니다.
