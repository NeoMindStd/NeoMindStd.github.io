# 무엇인가

Notepad++는 빠르고 가벼운 Windows용 텍스트 편집기입니다. IDE처럼 무겁지 않으면서도 구문 강조, 다중 탭, 플러그인, 검색/치환 기능이 잘 갖춰져 있어 로그 확인이나 설정 파일 편집에 특히 편합니다.

# 공식 사이트와 다운로드 경로

- 공식 사이트: [Notepad++](https://notepad-plus-plus.org/)
- 다운로드: [공식 다운로드 페이지](https://notepad-plus-plus.org/downloads/)

# 지원 OS와 설치 방법

## Windows

공식 다운로드 페이지에서 Windows용 설치 파일이나 패키지를 내려받아 설치합니다. 설치가 끝나면 첫 실행과 버전 확인을 한 번 진행하는 편이 좋습니다. 설치형과 포터블 중 환경에 맞는 쪽을 고르면 됩니다.

## macOS

현재 공식 macOS 배포를 확인하기 어렵습니다. 공식 macOS 빌드는 제공되지 않습니다.

## Linux

현재 공식 Linux 배포를 확인하기 어렵습니다. 공식 Linux 빌드는 제공되지 않습니다.

# 기본 사용법

1. 파일을 열고 언어 모드와 인코딩이 맞는지 먼저 확인합니다.
2. 검색/치환, 정규식, 컬럼 선택으로 반복 작업을 줄입니다.
3. 필요할 때 Plugins Admin에서 플러그인을 추가하되 호환성을 먼저 확인합니다.

# 주의할 점

인코딩과 줄바꿈 형식을 잘못 저장하면 파일이 깨질 수 있습니다. UTF-8과 LF/CRLF를 의식적으로 확인하는 습관이 중요합니다.

# 요약

Notepad++는 Windows에서 가장 실용적인 경량 편집기 중 하나입니다. 검색/치환과 플러그인 정도만 익혀도 체감 효율이 크게 올라갑니다.
