# 풀이
## 문제

두 세 자리 수를 거꾸로 읽은 값 중
더 큰 수를 출력하는 문제입니다.

## 코드

```python
a, b = input().split()
print(max(int(a[::-1]), int(b[::-1])))
```

## 설명

문자열 슬라이싱 `[::-1]`으로 숫자를 뒤집어 정수 변환 후
`max`를 출력합니다.
