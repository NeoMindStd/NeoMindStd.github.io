# 풀이

## 문제

구간 `[n, m]`의 정수 합을
시나리오 형식으로 출력하는 문제입니다.

## 코드

```python
for T in range(int(input())):
    n,m=map(int,input().split())
    print(f'Scenario #{T+1}:\n{(m*(m+1)-n*(n+1))//2+n}\n')
```

## 설명

등차수열 합 공식을 이용해
`1..m`에서 `1..(n-1)` 합을 빼는 방식으로 계산합니다.
