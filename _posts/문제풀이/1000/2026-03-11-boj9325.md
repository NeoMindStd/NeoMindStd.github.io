# 풀이

## 문제

기본 자동차 가격과 옵션 목록이 주어질 때
최종 가격을 계산하는 문제입니다.

## 코드

```python
for _ in range(int(input())):
    s = int(input())
    opt = [tuple(map(int, input().split())) for _ in range(int(input()))]
    v = sum(map(lambda t: t[0]*t[1], opt))
    print(s+v)
```

## 설명

각 옵션의 `수량 * 가격`을 전부 더해
기본 가격 `s`와 합산한 결과를 출력합니다.
