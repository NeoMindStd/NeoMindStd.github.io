# 풀이
## 문제

두 정수 `A`, `B`가 줄바꿈으로 주어질 때
합을 출력하는 문제입니다.

## 코드

```python
a = int(input())
b = int(input())

print(a+b)
```

## 설명

각 줄의 입력을 정수로 읽어
더한 값을 출력하면 됩니다.
