# 풀이

## 문제

두 수 `a`, `b`가 주어질 때
`a`가 `b`의 약수인지, 배수인지, 둘 다 아닌지 판별하는 문제입니다.

## 코드

```python
while True:
    a, b = map(int, input().split())
    if(a==b==0): break
    elif a%b==0: print("multiple")
    elif b%a==0: print("factor")
    else: print("neither")
```

## 설명

나머지 연산으로 배수/약수 관계를 검사하고
조건 우선순위에 따라 문자열을 출력합니다.
