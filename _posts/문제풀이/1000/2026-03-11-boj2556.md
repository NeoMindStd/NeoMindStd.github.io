# 풀이
## 문제

입력 `N`에 대해
`N x N` 크기의 별 사각형을 출력하는 문제입니다.

## 코드

```python
n = int(input())

for i in range(n):
    for j in range(n):
        print("*", end="")
    print()
```

## 설명

행과 열을 각각 `N`번 반복해
각 줄마다 별 `N`개를 출력하면 됩니다.
