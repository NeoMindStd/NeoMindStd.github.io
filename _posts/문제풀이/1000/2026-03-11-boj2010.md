# 풀이
## 문제

멀티탭 여러 개를 직렬로 연결할 때
최대로 꽂을 수 있는 전기기구 수를 구하는 문제입니다.

## 코드

```python
import sys

r = 1
for _ in range(int(sys.stdin.readline())):
    r += int(sys.stdin.readline()) - 1
print(r)
```

## 설명

처음 사용 가능한 구멍을 1로 두고,
각 멀티탭을 연결할 때마다
`(구멍 수 - 1)`만큼 순증가합니다.

모든 멀티탭 반영 후 `r`이 최대 기기 수입니다.
