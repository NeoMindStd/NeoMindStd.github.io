# 중기 5차: Tools 허브 구조 개편 (2026-03-17)

## 작업 목적

- Tools 아카이브를 방문자가 목적별로 더 빨리 고를 수 있는 구조로 바꾸기
- 개별 Tools 글 하단에서도 비슷한 도구 허브로 다시 이동할 수 있게 동선을 정리하기

## 반영 내용

### Tools 메인 허브

- `목적별 빠른 선택표` 추가
- `추천 경로` 추가
- 허브 구조를 중심으로 메인 진입 동선 정리

### 하위 허브 신설

- `/tools/development/` 개발·협업 도구 허브
- `/tools/system-utilities/` 시스템 점검·유틸리티 허브
- `/tools/creator-media/` 방송·오디오·영상 도구 허브

### 내부 링크 조정

- Tools 카테고리 포스트 하단 CTA를 개별 글 중심에서 허브 중심으로 변경
- 연재 목록에 새 Tools 허브 링크 추가

## 기대 효과

- Tools 메인 페이지에서 첫 클릭 결정 시간이 줄어든다
- 비슷한 유형의 도구를 연속으로 읽기 쉬워진다
- Search Console 데이터가 쌓이면 허브 단위로 랜딩 성과를 해석하기 쉬워진다
