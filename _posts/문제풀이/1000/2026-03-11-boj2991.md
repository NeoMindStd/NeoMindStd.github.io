# 풀이

## 문제

두 마리 개가 각각 공격/휴식 주기를 반복할 때,
세 사람이 도착한 시각에 몇 마리에게 공격당하는지 출력하는 문제입니다.

## 코드

```python
a,b,c,d=map(int,input().split())
l=list(map(int,input().split()))
for ll in l:
    s=0
    if 0!=ll%(a+b)<=a:s+=1
    if 0!=ll%(c+d)<=c:s+=1
    print(s)
```

## 설명

도착 시간을 각 주기 길이로 나눈 나머지가
공격 구간 안에 있으면 해당 개에게 공격당합니다.
두 개에 대해 각각 판별해 합산하면 됩니다.
