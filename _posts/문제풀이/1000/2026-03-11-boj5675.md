# 풀이

## 문제

입력으로 주어지는 값마다 조건을 만족하는지 판단해
`Y` 또는 `N`을 출력하는 문제입니다.

## 코드

```python
import sys

for a in sys.stdin:
    print('Y' if int(a)%6 == 0 else 'N')
```

## 설명

각 입력값이 `6`의 배수인지 검사해
배수면 `Y`, 아니면 `N`을 출력합니다.
