# 풀이
## 문제

입력된 수 `N`에 대해
`N x 1`부터 `N x 9`까지를 형식에 맞게 출력하는 문제입니다.

## 코드

```python
a = int(input())

for i in range(1, 10):
    print(a, "*" , i , "=" , a*i)
```

## 설명

반복문으로 `1..9`를 순회하며
곱셈식을 그대로 출력하면 됩니다.
