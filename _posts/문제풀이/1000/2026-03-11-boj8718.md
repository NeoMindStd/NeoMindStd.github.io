# 풀이

## 문제

예산 `x`와 기준값 `k`가 주어질 때,
가능한 후보 금액 중 예산 이하의 최대값을 찾는 문제입니다.

## 코드

```python
x,k=map(lambda n:int(n)*1000,input().split())
a=0
for i in [7*k//4,7*k//2,7*k]:
    if i<=x:
        a=i
print(a)
```

## 설명

문제에서 제시된 세 후보를 순서대로 확인해
예산 이하인 값 중 가장 큰 값을 갱신해 출력합니다.
