# 풀이
## 문제

`N!`의 뒤에서부터 연속된 0의 개수를 구하는 문제입니다.

## 코드

```python
def fact(n):return n*fact(n-1)if n>1 else 1
s=str(fact(int(input())))
for i in range(len(s)):
    if s[len(s)-i-1]!='0':break
print(i)
```

## 설명

팩토리얼 값을 문자열로 만든 뒤
뒤에서부터 처음 0이 아닌 문자를 만날 때까지 세어
0의 개수를 출력합니다.
