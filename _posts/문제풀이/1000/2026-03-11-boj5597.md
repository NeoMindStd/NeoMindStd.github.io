# 풀이

## 문제

1번부터 30번까지 학생 중
과제를 제출하지 않은 2명의 번호를 찾는 문제입니다.

## 코드

```python
stu = dict()

for i in range(1, 31):
    stu[i] = False

for _ in range(28):
    stu[int(input())] = True

cnt = 0
for i in range(1, 31):
    if not stu[i]:
       cnt += 1
       print(i)
       if cnt >= 2:
           break
```

## 설명

학생 번호별 제출 여부를 저장한 뒤,
제출되지 않은 번호를 순서대로 2개 출력합니다.
