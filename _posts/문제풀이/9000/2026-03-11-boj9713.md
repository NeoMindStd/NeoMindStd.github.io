# 풀이

## 문제

주어진 n에 대해 특정 홀수 수열의 합을 계산하는 문제입니다.

## 코드

```python
for T in range(int(input())):
    n=int(input())
    print((n+1)**2//4)

```

## 설명

홀수 합 공식을 이용하면 결과를 `(n+1)^2/4` 형태로 바로 계산할 수 있습니다.
