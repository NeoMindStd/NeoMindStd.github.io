# 풀이
## 문제

2007년의 월/일이 주어졌을 때
해당 날짜의 요일을 출력하는 문제입니다.

## 코드

```python
days = ['SUN', 'MON', 'TUE', 'WED', 'THU', 'FRI', 'SAT']

months = range(1, 13)
now = [1, 1]

mIn, dIn = input().split()
mIn = int(mIn)
dIn = int(dIn)

count = 1

pFlag = False;
for i in months:
    if(pFlag) :
        break
    
    now[0] = i
    
    if(i == 2):
        dd = range(1,29)
    elif ((i %2 == 0 and i < 7) or
        (i %2 != 0 and i > 7)):
        dd = range(1,31)
    else:
        dd = range(1,32)

    for j in dd:
        now[1] = j
        if(now[0] == mIn and now[1] == dIn):
            print(days[count%7])
            pFlag = True
            break
        count+=1
```

## 설명

2007-01-01부터 입력 날짜까지 날짜를 순회하며
요일 인덱스를 증가시켜 계산합니다.

목표 날짜에 도달했을 때 대응하는 요일 문자열을 출력합니다.
