# 풀이

## 문제

백준 13015번 문제를 풀이합니다.

## 코드

{% raw %}
`python
n = int(input())

print('*'*n + ' ' * (2*n-3) + '*'*n)
for i in range(1, n-1):
    print(' '*i + '*' + ' '*(n-2) + '*' + ' '*(2*(n-i)-3) + '*' + ' '*(n-2) + '*')
print(' '*(n-1) + '*' + ' '*(n-2) + '*' + ' '*(n-2) + '*')
for i in range(n-2, 0, - 1):
    print(' '*i + '*' + ' '*(n-2) + '*' + ' '*(2*(n-i)-3) + '*' + ' '*(n-2) + '*')
print('*'*n + ' ' * (2*n-3) + '*'*n)

```
