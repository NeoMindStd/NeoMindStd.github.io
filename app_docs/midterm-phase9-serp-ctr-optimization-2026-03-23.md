# 중기 9차: Search Console 기반 CTR 최적화 (2026-03-23)

## 배경

- 최근 30일 기준 Search Console 노출 `1,663`, 클릭 `13`, CTR `0.78%`
- 상위 노출 페이지 중 `/boj/`와 `/문제풀이/boj1186/`가 CTR 병목에 크게 기여

## 작업 목적

- 노출이 많은 페이지의 SERP 스니펫(제목/요약/설명) 품질을 개선해 클릭 전환율을 높이기
- 검색 랜딩 후 이탈을 줄이기 위해 초반 읽기 구조(빠른 답/다음 읽을거리)를 강화하기

## 반영 내용

### 1) `/boj/` 허브

- 제목/요약/설명을 검색 의도 중심으로 구체화
- 상단 `핵심 안내` 추가
  - 많이 찾는 문제로 즉시 이동
  - 번호 기반 탐색 경로를 빠르게 안내

### 2) `/문제풀이/boj1186/`

- 제목/요약/설명을 문제 핵심 키워드 중심으로 재작성
  - `N·K 조건`
  - `색칠한 면적 중 보이는 것의 최댓값`
  - `겹침 분할 구현`
- `빠른 답` 섹션 추가
- 요약/관련 글 블록 추가

## 기대 효과

- Search Console 고노출 페이지의 클릭 전환률 개선
- 검색 랜딩 직후 문제 핵심 파악 속도 개선
- 허브 재순환(boj 허브 -> 개별 문제 -> 유사 문제) 강화
