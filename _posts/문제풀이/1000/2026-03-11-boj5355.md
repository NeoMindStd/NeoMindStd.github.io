# 풀이

## 문제

기본 실수 값에 연산 기호(`@`, `%`, `#`)를 순서대로 적용해
최종 값을 소수 둘째 자리로 출력하는 문제입니다.

## 코드

```python
for T in range(int(input())):
    l = input().split()
    acc = float(l[0])

    for i in range(len(l)):
        if l[i] =='@': acc *= 3
        elif l[i] =='%': acc += 5
        elif l[i] =='#': acc -= 7

    print('%.2f' %acc)
```

## 설명

첫 토큰을 시작값으로 두고
연산 기호를 순회하며 누적 계산한 뒤 포맷 출력합니다.
