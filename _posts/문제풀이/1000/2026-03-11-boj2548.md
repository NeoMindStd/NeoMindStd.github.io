# 풀이
## 문제

수열이 주어질 때
전체 원소와의 거리 합이 최소가 되는 대표 자연수를 구하는 문제입니다.

## 코드

```python
n=int(input())
l=sorted(map(int,input().split()))
print(l[n//2-1])
```

## 설명

정렬 후 중앙값이 거리 합을 최소로 만듭니다.
문제 조건에 맞춰 `n//2-1` 위치 값을 출력합니다.
