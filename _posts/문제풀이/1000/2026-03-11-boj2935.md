# 풀이

## 문제

아주 큰 두 정수와 연산자(`+` 또는 `*`)가 주어질 때 결과를 출력하는 문제입니다.

## 코드

```python
import sys
a = int(sys.stdin.readline())
op = sys.stdin.readline()
b = int(sys.stdin.readline())
print(a*b if op[0]=='*' else a+b)
```

## 설명

파이썬 정수는 큰 자릿수도 그대로 처리할 수 있으므로,
연산자에 따라 덧셈 또는 곱셈만 분기하면 됩니다.
