# 풀이

## 문제

세 막대 길이로
직각삼각형인지, 정삼각형인지, 그 외인지 판별하는 문제입니다.

## 코드

```python
a,b,c=tuple(sorted(map(int,input().split())))
if a*a+b*b==c*c:print(1)
elif a==c:print(2)
else: print(0)
```

## 설명

길이를 정렬한 뒤 피타고라스 조건을 먼저 검사하고,
아니면 세 변 동일 여부를 확인해 분류합니다.
