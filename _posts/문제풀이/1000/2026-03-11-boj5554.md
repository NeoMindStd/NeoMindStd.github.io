# 풀이

## 문제

네 구간 이동 시간을 모두 더해
분과 초 형태로 출력하는 문제입니다.

## 코드

```python
t = sum([int(input()) for _ in range(4)])
print("%d\n%d" %(t//60,t%60))
```

## 설명

총 초를 구한 뒤 `// 60`으로 분,
`% 60`으로 초를 계산해 줄바꿈 출력합니다.
