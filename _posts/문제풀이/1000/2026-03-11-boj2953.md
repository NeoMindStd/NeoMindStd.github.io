# 풀이

## 문제

5명의 참가자가 받은 점수 합을 계산해서
가장 높은 점수를 받은 참가자 번호와 점수를 출력하는 문제입니다.

## 코드

```python
print(*sorted([(i,sum(map(int,input().split())))for i in range(1,6)],key=lambda x:x[1])[-1])
```

## 설명

각 참가자에 대해 `(번호, 합계)`를 만들고,
합계 기준 정렬 후 마지막 원소를 출력하면 최대 점수 정보를 얻을 수 있습니다.
