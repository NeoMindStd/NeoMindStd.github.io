# 풀이

## 문제

돌 `n`개에서 1개 또는 3개를 번갈아 가져갈 때
선공/후공 중 누가 이기는지 판별하는 문제입니다.

## 코드

```python
print('SK'if int(input())%2==0 else'CY')
```

## 설명

`n`이 짝수면 선공(SK), 홀수면 후공(CY)이 이깁니다.
