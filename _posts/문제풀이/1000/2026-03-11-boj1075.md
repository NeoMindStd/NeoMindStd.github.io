# 풀이
## 문제

정수 `N`의 마지막 두 자리를 `00`으로 바꾼 뒤,
어떤 수 `F`로 나누어떨어지게 만드는 가장 작은 두 자리 수를 구하는 문제입니다.

## 코드

```python
n, f = int(input()), int(input()) 
p = (n//100) * 100
for i in range(p, p+100):
    if i % f == 0:
        print("%02d" %(i%100))
        break
```

## 설명

`N`의 마지막 두 자리를 0으로 만든 값 `p`부터 `p+99`까지 순회하면서
`F`로 나누어떨어지는 첫 값을 찾습니다.

그 값의 마지막 두 자리를 두 자리 형식(`%02d`)으로 출력하면 됩니다.
