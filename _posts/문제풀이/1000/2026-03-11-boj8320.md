# 풀이

## 문제

면적이 `1`부터 `n`까지인 직사각형의
서로 다른 `(가로, 세로)` 조합 개수를 구하는 문제입니다.

## 코드

```python
n=int(input())
s=0
for i in range(1, n+1):
    for j in range(1, int(i**.5)+1):
        if i%j==0: s+=1
print(s)
```

## 설명

각 면적 `i`에 대해 `j <= sqrt(i)` 범위의 약수 개수를 세면
중복 없는 직사각형 개수를 구할 수 있습니다.
