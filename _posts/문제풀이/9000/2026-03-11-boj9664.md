# 풀이

## 문제

주어진 규칙에 맞춰 최소값과 최대값을 계산해 출력하는 문제입니다.

## 코드

```python
n,o=int(input()),int(input())
m=o//(n-1)
print(o+m-(1 if m*(n-1)==o else 0), o+m)

```

## 설명

입력된 두 수의 관계를 이용해 몫을 계산하고, 경계 조건에 따라 최소/최대 값을 산출합니다.
