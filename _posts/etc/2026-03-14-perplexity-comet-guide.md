# 어떤 서비스인가

Perplexity Comet은 검색과 브라우징, 요약, 후속 행동 제안을 한 브라우저 안에 묶으려는 서비스입니다. 그냥 검색창에 답만 붙이는 타입이 아니라, 브라우저 자체를 AI 중심 인터페이스로 바꾸는 쪽에 가깝습니다.

기존 모델 비교 관점이 먼저 필요하면 [2026 프론티어 모델 비교](/기타/frontier-models-comparison-2026/)나 [딥리서치란 무엇인가](/기타/what-is-deep-research/) 글을 먼저 보고 오는 것도 좋습니다.

# 공식 사이트와 다운로드 경로

- 공식 사이트: [Perplexity Comet](https://www.perplexity.ai/comet)
- 다운로드/접속 경로: [Get started with Comet](https://www.perplexity.ai/help-center/en/articles/11172798-getting-started-with-comet)
- 공식 안내: [Comet 소개 페이지](https://www.perplexity.ai/comet)

# 지원 환경과 설치 방법

공식 도움말 기준으로 Comet for Desktop은 현재 Windows와 macOS를 대상으로 안내됩니다. 같은 문서에서 iOS와 Android 기기에서는 이 데스크톱 설치 방식이 아직 지원되지 않는다고 설명합니다.

## Windows

Windows에서는 Chrome이나 Edge 같은 Chromium 계열 브라우저에서 Comet 설치를 시작하는 흐름이 가장 대표적입니다. 설치가 끝나면 Perplexity 계정으로 로그인하고 기본 브라우저 전환 여부를 확인하는 단계가 이어집니다.

1. [Comet 페이지](https://www.perplexity.ai/comet) 또는 Help Center 안내에서 설치를 시작합니다.
2. Chrome 또는 Edge에서 설치 절차를 진행합니다.
3. 로그인 후 기본 브라우저 설정과 단축키, 기본 검색 흐름을 확인합니다.

## macOS

macOS도 공식 도움말상 Windows와 같은 데스크톱 설치 경로를 따릅니다. 핵심은 "브라우저 확장 하나 더"가 아니라, Comet 자체를 브라우저로 쓰는 경험에 가깝다는 점입니다.

1. [Comet 소개 페이지](https://www.perplexity.ai/comet)에서 설치를 시작합니다.
2. 로그인 후 Perplexity 계정과 브라우징 권한을 연결합니다.
3. 단축키와 기본 브라우저 동작을 자신 취향에 맞게 손봅니다.

# 기본 사용법

Comet을 처음 쓸 때는 "질문 -> 검색 결과 -> 새 탭 정리" 흐름보다, 지금 보고 있는 문맥을 AI에게 이해시키는 감각으로 접근하는 편이 좋습니다. 공식 설명도 검색, 브라우징, 요약, 비교를 한 흐름으로 연결하는 쪽에 초점이 있습니다.

1. 현재 보고 있는 페이지를 요약해 달라고 요청합니다.
2. 같은 주제의 다른 페이지나 상품을 비교해 달라고 이어서 말합니다.
3. 관련 탭을 열거나 후속 탐색 질문을 던져 한 세션 안에서 흐름을 이어갑니다.

# 어떤 사람에게 잘 맞나

Comet은 검색창을 자주 왔다 갔다 하는 사람, 비교 쇼핑이나 리서치 탭이 늘 많은 사람에게 특히 잘 맞습니다. 반대로 브라우저 자체는 최대한 가볍고, AI는 필요할 때만 붙이고 싶은 사람이라면 [Brave Leo 사용법](/기타/brave-leo-guide/)이나 [Copilot Mode in Edge 사용법](/기타/edge-copilot-mode-guide/) 쪽이 더 잘 맞을 수 있습니다.

# 주의할 점

Comet은 일반 브라우저에 AI 버튼을 하나 더 얹은 수준이 아니라 브라우저 경험 자체를 바꾸는 타입이라, 계정 연동과 기본 브라우저 설정, 데이터 처리 감각을 먼저 확인하는 편이 좋습니다. 또 공식 안내상 지원 범위가 데스크톱 중심이라 모바일까지 같은 흐름을 기대하면 어긋날 수 있습니다.

# 요약

Perplexity Comet은 "검색을 잘하는 AI"에서 한 걸음 더 나아가, 브라우저 안에서 탐색과 요약, 후속 행동까지 이어 주려는 제품입니다. 2026년 기준으로는 Windows와 macOS 데스크톱 중심으로 접근하는 편이 가장 자연스럽고, 처음에는 작은 조사 작업부터 맡기면서 감을 잡는 게 좋습니다.
