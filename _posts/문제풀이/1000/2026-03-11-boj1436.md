# 풀이
## 문제

숫자에 `"666"`이 포함되는 수를 순서대로 나열할 때
`N`번째 수를 구하는 문제입니다.

## 코드

```python
n = int(input())

cnt = 0

i = 666
while True:
    if str(i).find("666") != -1:
        cnt += 1
    if cnt == n:
        print(i)
        break
    i += 1
```

## 설명

`666`부터 시작해 1씩 증가시키며
문자열에 `"666"`이 포함되면 카운트합니다.

카운트가 `N`이 되는 순간의 수를 출력합니다.
