# 풀이

## 문제

`n x m` 격자에서
둘레 조건을 만족하는 직사각형 개수를 구하는 문제입니다.

## 코드

```python
n,m,p=map(int,input().split())
r=0
for i in range(n):
    for j in range(m):
        if (i+j+2)*2>=p:r+=(n-i)*(m-j)
print(r)
```

## 설명

좌상단 좌표 기준으로 가능한 모든 우하단 선택 개수를 더하면서,
문제 조건 `(i+j+2)*2 >= p`를 만족할 때만 누적합니다.
