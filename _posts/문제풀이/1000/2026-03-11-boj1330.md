# 풀이
## 문제

두 정수 `A`, `B`를 비교해
`>`, `<`, `==` 중 하나를 출력하는 기본 비교 문제입니다.

## 코드

```python
a, b = tuple(map(int, input().split()))
print(">" if a > b else "<" if a < b else "==")
```

## 설명

삼항 연산자로 조건 분기를 한 줄로 처리했습니다.

 - `a > b` 이면 `>`
 - `a < b` 이면 `<`
 - 그 외는 `==`
