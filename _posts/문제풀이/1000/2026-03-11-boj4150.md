# 풀이

## 문제

정수 `n`번째 피보나치 수를 출력하는 문제입니다.

## 코드

```python
n = int(input())

fibo = [2, 1, 1]

while fibo[0] < n:
    fibo.append(fibo[-1]+fibo[-2])
    fibo[0]+=1

print(fibo[n])
```

## 설명

리스트에 피보나치 값을 순차적으로 채워 넣고
요청 인덱스 `n` 위치 값을 출력합니다.
