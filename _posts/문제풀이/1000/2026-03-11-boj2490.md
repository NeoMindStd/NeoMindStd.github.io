# 풀이
## 문제

세 번의 입력에서
윷가락 뒤집힌 면(1) 개수에 따라
`A, B, C, D, E` 중 해당 결과를 출력하는 문제입니다.

## 코드

```python
for _ in range(3):
    s = sum(map(int, input().split()))
    c = ""
    if s == 0:
        c = "D"
    elif s==1:
        c = "C"
    elif s==2:
        c = "B"
    elif s==3:
        c = "A"
    else:
        c = "E"
    print(c)
```

## 설명

합계 `s`가 뒤집힌 면 개수이므로
문제 표에 맞게 문자로 매핑해 출력하면 됩니다.
