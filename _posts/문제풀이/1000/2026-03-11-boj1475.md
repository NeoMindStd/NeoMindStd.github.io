# 풀이
## 문제

주어진 방 번호를 만들기 위해 필요한 숫자 세트의 최소 개수를 구하는 문제입니다.
숫자 `6`과 `9`는 서로 바꿔 쓸 수 있습니다.

## 코드

```python
n = list(map(int, input()))

a = [0 for _ in range(10)]

while(True):
    if(len(n)==0) :
        break
    a[n[-1]]+=1
    del n[-1]

a[6]=a[9]=(a[6]+a[9])//2 + (a[6]+a[9])%2
print(max(a))
```

## 설명

각 숫자 등장 횟수를 센 뒤,
`6`과 `9`는 합쳐서 `ceil((count6 + count9)/2)`로 처리합니다.

그 후 가장 많이 필요한 숫자의 개수가
최소 세트 수가 됩니다.
