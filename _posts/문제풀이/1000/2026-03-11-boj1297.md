# 풀이
## 문제

대각선 길이 `D`와 화면 비율 `H:W`가 주어질 때
실제 높이와 너비를 구하는 문제입니다.

## 코드

```python
a,b,c=map(int,input().split())
x=(a*a/(b*b+c*c))**0.5
print(int(x*b), int(x*c))
```

## 설명

높이와 너비를 각각 `x*b`, `x*c`라고 두면
피타고라스 정리에 의해

`(x*b)^2 + (x*c)^2 = D^2`

이므로 `x = sqrt(D^2 / (b^2 + c^2))`를 구해
정수 부분만 출력합니다.
