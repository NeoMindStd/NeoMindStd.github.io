# 풀이
## 문제

두 수에서 `5`와 `6`이 서로 헷갈릴 수 있을 때
가능한 최소 합과 최대 합을 구하는 문제입니다.

## 코드

```python
a, b = map(str, input().split())
print(int(a.replace('6', '5')) + int(b.replace('6', '5')),
      int(a.replace('5', '6')) + int(b.replace('5', '6')))
```

## 설명

최소 합은 모든 `6`을 `5`로 바꿔 계산하고,
최대 합은 모든 `5`를 `6`으로 바꿔 계산합니다.
