# 풀이
## 문제

16진수 문자열을 10진수 정수로 변환해 출력하는 문제입니다.

## 코드

```python
print(int(input(),16))
```

## 설명

`int(문자열, 16)`을 사용해 16진수 입력을 정수로 바로 파싱합니다.
