# 풀이

## 문제

사탕 총합과 두 사람 차이가 주어질 때,
각각의 사탕 개수를 구하는 문제입니다.

## 코드

```python
a,b=int(input()),int(input())
print((a+b)//2)
print((a-b)//2)
```

## 설명

`x+y=a`, `x-y=b` 연립식에서
`x=(a+b)/2`, `y=(a-b)/2`를 계산해 출력합니다.
