# 풀이

## 문제

라운드별 점수를 비교해
두 플레이어의 승리 횟수를 출력하는 문제입니다.

## 코드

```python
import sys
read=sys.stdin.readline
s=[0,0]
for _ in range(int(read())):
    a,b=map(int,read().split())
    if a>b:s[0]+=1
    if a<b:s[1]+=1
print(*s)
```

## 설명

각 라운드마다 큰 점수를 낸 쪽 카운트를 증가시키고
최종 승수를 출력합니다.
