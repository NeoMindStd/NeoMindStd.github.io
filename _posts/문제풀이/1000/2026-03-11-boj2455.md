# 풀이
## 문제

4개 역에서
내린 사람 수와 탄 사람 수가 순서대로 주어질 때
기차 안 승객 수의 최댓값을 구하는 문제입니다.

## 코드

```python
x = 0
m = 0
for _ in range(4):
    a, b = map(int, input().split())
    x -= a
    m = max(m, x)
    x += b
    m = max(m, x)
print(m)
```

## 설명

현재 승객 수 `x`를 역마다 갱신하면서
매 단계 `max`로 최댓값을 추적하면 정답을 구할 수 있습니다.
