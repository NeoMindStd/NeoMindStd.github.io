# 풀이

## 문제

반지름 `r`이 주어질 때
유클리드 기하학 원의 넓이와 택시 기하학 원의 넓이를 출력하는 문제입니다.

## 코드

```python
r = int(input())
print(r*r*3.141592653589)
print(2*r*r)
```

## 설명

유클리드 원 넓이는 `πr²`,
택시 기하학 원 넓이는 `2r²` 공식을 그대로 사용합니다.
