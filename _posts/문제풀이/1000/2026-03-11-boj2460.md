# 풀이
## 문제

10개 역에서
내린 사람 수와 탄 사람 수가 주어질 때
열차 내 승객 수의 최댓값을 구하는 문제입니다.

## 코드

```python
x = 0
m = 0
for _ in range(10):
    a, b = map(int, input().split())
    x -= a
    m = max(m, x)
    x += b
    m = max(m, x)
print(m)
```

## 설명

현재 승객 수를 `x`로 유지하면서
역마다 하차/승차를 반영하고,
각 단계에서 최댓값 `m`을 갱신하면 됩니다.
