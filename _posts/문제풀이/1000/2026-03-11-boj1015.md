# 풀이
## 문제

수열 `A`를 정렬해 `B`를 만들 때
`B[P[i]] = A[i]`를 만족하는 순열 `P`를 구하는 문제입니다.

## 코드

```python
input();print(*sorted(map(lambda x:int(x)-1,input().split())))
```

## 설명

해당 코드에서는 입력된 값이 `1..N`의 순열 형태라는 전제에서
각 원소를 `-1` 변환 후 정렬해 바로 출력합니다.
