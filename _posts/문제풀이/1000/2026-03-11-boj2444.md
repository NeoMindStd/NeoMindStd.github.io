# 풀이
## 문제

입력 `N`에 대해
위로 커졌다가 아래로 다시 줄어드는 다이아몬드 형태의 별을 출력하는 문제입니다.

## 코드

```python
n = int(input())

for i in range(n):
    for j in range(n-1-i):
        print(" ", end="")
    for j in range(2*(i+1)-1):
        print("*", end="")
    print()
    
for i in range(1, n):
    for j in range(i):
        print(" ", end="")
    for j in range(2*((n-i))-1):
        print("*", end="")
    print()
```

## 설명

윗부분은 2442번 패턴,
아랫부분은 2443번 패턴을 이어 붙이면 됩니다.
