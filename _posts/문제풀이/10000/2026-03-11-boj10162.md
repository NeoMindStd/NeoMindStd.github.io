# 풀이

## 문제

백준 10162번 문제를 풀이합니다.

## 코드

{% raw %}
`python
n = int(input())

a, b, c = 300, 60, 10
if  n%c!=0:
    print(-1)
else:
    print(n//a, (n%a)//b, (n%b)//c)

```
{% endraw %}
