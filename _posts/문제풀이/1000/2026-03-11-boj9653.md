# 풀이

## 문제

정해진 스타워즈 ASCII 아트를
그대로 출력하는 문제입니다.

## 코드

```python
print("""    8888888888  888    88888
   88     88   88 88   88  88
    8888  88  88   88  88888
       88 88 888888888 88   88
88888888  88 88     88 88    888888

88  88  88   888    88888    888888
88  88  88  88 88   88  88  88
88 8888 88 88   88  88888    8888
 888  888 888888888 88  88      88
  88  88  88     88 88   88888888""")
```

## 설명

입력 없이 멀티라인 문자열을
정확히 출력하면 정답입니다.
