# 풀이

## 문제

각 사람의 몸무게와 키를 비교해
자신보다 덩치 큰 사람 수에 따라 등수를 매기는 문제입니다.

## 코드

```python
n = int(input())

people = list()
ranks = [1]*n

for i in range(n):
    people.append(tuple(map(int, input().split())))

for i in range(len(people)):
    for j in range(len(people)):
        if people[i][0] > people[j][0] and people[i][1] > people[j][1]:
            ranks[j] += 1

print(*ranks)
```

## 설명

모든 사람 쌍을 비교해서
둘 다 더 큰 사람이 있으면 상대의 등수를 1 증가시킵니다.
