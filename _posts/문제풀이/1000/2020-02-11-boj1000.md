# 풀이

## 문제
두 정수 A와 B를 입력받아 합을 출력

## 코드

```c
#include <stdio.h>

int main() {
    int a, b;
    scanf("%d %d", &a, &b);
    printf("%d", a+b);
    
    return 0;
}
```

## 설명
백준에서 가장 간단한 문제 중 하나라서, C 입출력 문법을 처음 익힐 때 가장 많이 거치는 문제입니다.

입출력 조건에는 나와있지 않지만, A와 B는 정수입니다.

핵심은 두 개의 정수 변수를 선언한 뒤 `scanf`로 입력받고, `printf`로 두 수의 합을 출력하는 것입니다.

시간복잡도는 `O(1)`이고 메모리 사용도 사실상 상수 수준입니다.
