# 풀이

## 문제

국어/수학 숙제를 각각 하루 처리량으로 나눠 필요한 일수를 구하고,
둘 중 더 오래 걸리는 과목 기준으로 방학 중 남는 날짜를 계산하는 문제입니다.

## 코드

```python
import math
l=[int(input()) for _ in range(5)]
print(l[0]-max(math.ceil(l[1]/l[3]), math.ceil(l[2]/l[4])))
```

## 설명

전체 방학일에서 `max(국어 필요일, 수학 필요일)`을 빼면 정답입니다.
나눗셈은 올림 처리가 필요하므로 `math.ceil`을 사용합니다.
