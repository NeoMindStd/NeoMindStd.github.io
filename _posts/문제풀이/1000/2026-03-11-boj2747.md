# 풀이
## 문제

정수 `N`이 주어질 때
`N`번째 피보나치 수를 출력하는 문제입니다.

## 코드

```python
n = int(input())

fibo = [0, 1]
while len(fibo) <= n:
    fibo.append(fibo[-1] + fibo[-2])

print(fibo[-1])
```

## 설명

피보나치 수열을 리스트에 차례대로 쌓아
인덱스 `N`까지 채운 뒤 출력합니다.
