# 풀이
## 문제

초기 치킨 수 `n`과 쿠폰 교환 기준 `k`가 주어질 때,
쿠폰 교환을 반복해 먹을 수 있는 총 치킨 수를 구하는 문제입니다.

## 코드

```python
import sys
for line in sys.stdin:
    n,k=map(int,line.split())
    t=n
    while t>=k:
        n+=t//k
        t=t//k+t%k
    print(n)
```

## 설명

`t`를 현재 쿠폰 수로 두고
쿠폰이 `k` 이상일 때마다

 - `t//k`마리 추가 구매
 - 남은 쿠폰 `t%k`와 새 쿠폰을 합침

을 반복해 총 치킨 수를 계산합니다.
