# 풀이
## 문제

고정비 `A`, 가변비 `B`, 판매가 `C`가 주어질 때
손익분기점(처음으로 이익이 나는 판매량)을 구하는 문제입니다.

## 코드

```python
a, b, c = tuple(map(int, input().split()))
print(a // (c-b) + 1 if b<c else -1)
```

## 설명

`B >= C`면 팔수록 이익이 없으므로 `-1`입니다.

그 외에는
`A + Bx < Cx`를 만족하는 최소 정수 `x`가 답이며,
`x = A // (C-B) + 1`로 계산합니다.
