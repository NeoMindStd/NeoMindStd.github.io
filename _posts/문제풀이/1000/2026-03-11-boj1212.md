# 풀이
## 문제

8진수 문자열이 주어질 때
이를 2진수 문자열로 변환해 출력하는 문제입니다.

## 코드

```python
print(format(int(input(), 8), 'b'))
```

## 설명

`int(input(), 8)`로 8진수를 10진수 정수로 파싱한 뒤,
`format(..., 'b')`로 2진수 문자열로 변환해 출력합니다.
