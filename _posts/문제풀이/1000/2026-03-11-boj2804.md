# 풀이
## 문제

두 단어가 주어질 때
처음으로 만나는 공통 문자를 기준으로 십자 형태를 출력하는 문제입니다.

## 코드

```python
a,b=input().split()
s=[0,0]

for i in range(len(a)):
    try:
        s[1]=b.index(a[i])
        s[0]=i
        break
    except:pass

for i in range(len(b)):
    if i==s[1]:o=a
    else:
        o=''
        for j in range(len(a)):
            if j==s[0]:o+=b[i]
            else:o+='.'
    print(o)
```

## 설명

공통 문자의 위치를 찾은 뒤
교차 행에는 가로 단어를, 나머지 행에는 교차 열 문자만 배치합니다.
