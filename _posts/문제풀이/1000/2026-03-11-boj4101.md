# 풀이

## 문제

여러 쌍의 정수 `a, b`에 대해
`a > b`이면 `Yes`, 아니면 `No`를 출력하는 문제입니다.

## 코드

```python
while True:
    a,b=map(int,input().split())
    if a==b==0:break
    print('Yes'if a>b else'No')
```

## 설명

`0 0`이 입력될 때까지 반복하며
대소 비교 결과를 바로 출력하면 됩니다.
