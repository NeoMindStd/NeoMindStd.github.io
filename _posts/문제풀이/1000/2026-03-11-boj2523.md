# 풀이
## 문제

별 개수가 `1`부터 `N`까지 늘었다가
다시 `N-1`부터 `1`까지 줄어드는 패턴을 출력하는 문제입니다.

## 코드

```python
n = int(input())

for i in range(n):
    for j in range(i+1):
        print('*', end='')
    print()
    
for i in range(1, n):
    for j in range(n-i):
        print('*', end='')
    print()
```

## 설명

증가 구간과 감소 구간을 분리해
각 줄의 별 개수만 정확히 맞춰 출력하면 됩니다.
