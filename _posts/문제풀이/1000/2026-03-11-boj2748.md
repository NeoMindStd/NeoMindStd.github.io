# 풀이
## 문제

큰 `N` 범위에서
`N`번째 피보나치 수를 출력하는 문제입니다.

## 코드

```python
fibo = [0, 1]
for i in range(2, 91):
    fibo.append(fibo[i-2] + fibo[i-1])

print(fibo[int(input())])
```

## 설명

문제의 최대 범위(`N <= 90`)까지
미리 DP 배열을 계산해 두고
해당 인덱스를 출력합니다.
