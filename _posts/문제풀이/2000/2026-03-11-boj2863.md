# 풀이

## 문제

2x2 배열을 회전해 식 값이 최소가 되는 회전 횟수를 구하는 문제입니다.

## 코드

```python
a, b = map(int, input().split())
c, d = map(int, input().split())

nums = [c*d, d*b, b*a, a*c]
print(nums.index(min(nums)))
```

## 설명

문제 식에 해당하는 4가지 회전 결과를 계산해 최솟값 인덱스를 출력합니다.
