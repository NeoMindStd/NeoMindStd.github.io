# 풀이

## 문제

`N`번 자를 때 만들 수 있는 최대 조각 수를 구하는 문제입니다.

## 코드

```python
n=int(input())
a=n//2
print((a+1)**2+(a+1 if n%2==1 else 0))
```

## 설명

가로/세로 절단 횟수를 최대한 균등하게 나누면 조각 수가 최대가 됩니다.
코드는 이 공식을 그대로 계산합니다.
