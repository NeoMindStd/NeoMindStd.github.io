# 풀이
## 문제

행렬 `A(NxM)`과 `B(MxK)`가 주어질 때
곱 행렬 `C(NxK)`를 출력하는 문제입니다.

## 코드

```python
n, m = map(int, input().split())
a = []
for _ in range(n):
    a.append(list(map(int, input().split())))
_, k = map(int, input().split())
b = []
for _ in range(m):
    b.append(list(map(int, input().split())))

for i in range(n):
    for g in range(k):
        tmp = 0
        for j in range(m):
            tmp += a[i][j] * b[j][g]
        print(tmp, end=" ")
    print()
```

## 설명

곱셈 정의에 따라
`C[i][g] = sum(A[i][j] * B[j][g])`를 계산해 출력합니다.
