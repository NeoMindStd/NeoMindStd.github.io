# 무엇인가

Hitomi Downloader는 다양한 사이트의 파일을 대기열 기반으로 관리하는 데스크톱 유틸리티입니다. 다만 어떤 자료를 내려받는지는 권한과 약관 문제를 크게 타므로, 합법적인 범위에서만 사용하는 전제가 필요합니다.

# 공식 사이트와 다운로드 경로

- 공식 사이트: [Hitomi Downloader](https://github.com/KurtBestor/Hitomi-Downloader)
- 다운로드: [공식 다운로드 페이지](https://github.com/KurtBestor/Hitomi-Downloader/releases)

# 지원 OS와 설치 방법

## Windows

공식 다운로드 페이지에서 Windows용 설치 파일이나 패키지를 내려받아 설치합니다. 설치가 끝나면 첫 실행과 버전 확인을 한 번 진행하는 편이 좋습니다. 공식 GitHub Releases만 기준으로 설치하고, 실행 전 릴리스 주체와 파일 무결성을 먼저 확인하는 편이 좋습니다.

# 기본 사용법

1. 프로그램을 실행한 뒤 저장 폴더와 파일명 규칙부터 정합니다.
2. URL이나 항목을 대기열에 추가하고 필요한 품질과 형식을 검토합니다.
3. 작업이 끝나면 로그와 저장 경로를 확인해 누락과 중복을 점검합니다.

# 주의할 점

도구 사용법보다 더 중요한 건 자료 접근 권한입니다. 저작권과 서비스 약관을 위반하는 용도로는 사용하지 않는 것이 맞고, 업무용이라면 저장 구조를 더 엄격히 잡는 편이 좋습니다.

# 요약

Hitomi Downloader는 Windows 중심의 대기열 다운로드 유틸리티입니다. 공식 GitHub 저장소와 릴리스만 기준으로 접근하고, 권한 있는 자료에만 사용하는 것이 핵심입니다.
