# 풀이
## 문제

두 자연수의
최대공약수(GCD)와 최소공배수(LCM)를 구하는 문제입니다.

## 코드

```python
a, b = input().split()
a = int(a)
b = int(b)

def gcd(a, b):
    r = b % a
    if r == 0:
        return a
    else:
        return gcd(r, a)

def lcm(a,b) :
    return (a*b)//gcd(a,b)

print(gcd(a,b))
print(lcm(a,b))
```

## 설명

유클리드 호제법으로 최대공약수를 구하고,
`LCM = (a*b) / GCD` 공식을 사용해 최소공배수를 계산합니다.
