# 풀이

## 문제

특정 문자 조합(크로아티아 알파벳)을 하나의 글자로 취급할 때
전체 글자 수를 구하는 문제입니다.

## 코드

```python
ca = ['c=', 'c-', 'dz=', 'd-', 'lj', 'nj', 's=', 'z=']
s = input()

for ch in ca:
    s = s.replace(ch, 'a')

print(len(s))
```

## 설명

특수 조합을 모두 한 글자 대체 문자로 바꾼 뒤
문자열 길이를 세면 정답이 됩니다.
