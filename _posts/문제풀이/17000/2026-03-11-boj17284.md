# 풀이

## 문제

백준 17284번 문제를 풀이합니다.

## 코드

{% raw %}
`python
l=map(int,input().split())
s=5000
for i in l:
    if i<2:s-=500
    elif i>2:s-=1000
    else:s-=800
print(s)

```
