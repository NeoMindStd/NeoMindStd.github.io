# 풀이

## 문제

두 팀 점수의 합과 차가 주어질 때
실제 점수 쌍을 구하는 문제입니다.

## 코드

```python
apb,amb=map(int,input().split())
a=(apb+amb)//2
b=apb-a
if b>=0 and a<=1000 and a+b==apb and a-b==amb: print(*sorted([a,b],reverse=True))
else: print(-1)
```

## 설명

연립식으로 점수를 계산한 뒤
음수 여부와 합/차 일치 여부를 검증해 가능하면 점수를, 아니면 `-1`을 출력합니다.
