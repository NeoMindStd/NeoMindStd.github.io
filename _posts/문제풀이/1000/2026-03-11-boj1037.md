# 풀이
## 문제

진짜 약수(1과 자기 자신 제외)들이 주어질 때
원래 수 `N`을 구하는 문제입니다.

## 코드

```python
input()
nums = list(map(int, input().split()))
print(min(nums)*max(nums))
```

## 설명

약수를 정렬했을 때
가장 작은 약수와 가장 큰 약수의 곱이 원래 수 `N`이 됩니다.

그래서 `min * max`를 출력하면 됩니다.
