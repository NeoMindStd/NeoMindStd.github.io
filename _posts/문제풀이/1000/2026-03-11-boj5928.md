# 풀이

## 문제

기준 시각(11일 11:11)부터
주어진 시각까지 지난 분(minute)을 구하는 문제입니다.
기준보다 빠르면 `-1`을 출력합니다.

## 코드

```python
h,d,m=map(int,input().split())
t=h*1440+d*60+m-16511
print(-1 if t<0 else t)
```

## 설명

날짜, 시, 분을 분 단위로 환산해
기준 분(`16511`)을 뺀 값을 출력합니다.
