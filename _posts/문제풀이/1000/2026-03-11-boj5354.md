# 풀이

## 문제

크기 `n`에 대해 테두리는 `#`, 내부는 `J`로 구성된 박스를 출력하는 문제입니다.

## 코드

```python
l=[]
for T in range(int(input())):
    n=int(input())
    r=('#'*n+'\n'+f'#{"J"*(n-2)}#\n'*(n-2)+'#'*n) if n>1 else '#'
    l.append(r)
print('\n\n'.join(l))
```

## 설명

`n==1` 예외를 분기하고,
나머지는 문자열 조합으로 윗줄/중간줄/아랫줄을 생성해 출력합니다.
