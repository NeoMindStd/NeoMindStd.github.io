# 풀이
## 문제

문서 문자열에서 단어가 겹치지 않게 등장하는 최대 횟수를 구하는 문제입니다.

## 코드

```python
s = input()
word = input()
cnt = 0
while(True):
    index = s.find(word)
    if(index == -1):
        break
    s = s[s.find(word)+len(word):]
    cnt += 1
print(cnt)
```

## 설명

`find`로 단어의 첫 등장 위치를 찾고,
찾았다면 그 단어 길이만큼 앞부분을 잘라낸 뒤 카운트합니다.

이 과정을 반복하면 겹치지 않는 최대 등장 횟수를 얻을 수 있습니다.
