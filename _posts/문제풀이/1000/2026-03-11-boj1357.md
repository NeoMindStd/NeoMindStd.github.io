# 풀이
## 문제

`Rev(Rev(X) + Rev(Y))` 값을 구하는 문제입니다.

## 코드

```python
x, y = map(int, input().split())

def rev(n):
    n = list(str(n))
    n.reverse()
    return int("".join(map(str, n)))

print(rev(rev(x)+rev(y)))
```

## 설명

`rev` 함수로 숫자를 문자열 뒤집기 후 정수로 변환합니다.

 - `X`, `Y`를 각각 뒤집음
 - 더한 값을 다시 뒤집음

최종 결과를 출력합니다.
