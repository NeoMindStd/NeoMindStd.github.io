# 풀이
## 문제

영문 문자열이 주어질 때
소문자는 대문자로, 대문자는 소문자로 바꿔 출력하는 문제입니다.

## 코드

```python
print(input().swapcase())
```

## 설명

문자열 메서드 `swapcase()`를 사용하면
대소문자 변환을 한 번에 처리할 수 있습니다.
