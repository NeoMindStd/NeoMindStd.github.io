# 풀이

## 문제

문제에서 제시한 ASCII 아트를 그대로 출력하는 문제입니다.

## 코드

```python
print("""       _.-;;-._
'-..-'|   ||   |
'-..-'|_.-;;-._|
'-..-'|   ||   |
'-..-'|_.-''-._|""")
```

## 설명

멀티라인 문자열을 그대로 출력하면 정답 형식을 만족합니다.
