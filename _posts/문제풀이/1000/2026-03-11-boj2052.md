# 풀이
## 문제

입력 `n`에 대해 `2^-n` 값을 소수 형태로 출력하는 문제입니다.

## 코드

```python
a="%.250f"%(2**-int(input()))
c=0
for i in range(len(a)):
    if a[len(a)-i-1]=='0':
        c+=1
    else:break
print(a[:len(a)-c])
```

## 설명

충분히 긴 소수 자리로 포맷한 뒤,
뒤쪽 불필요한 0을 제거해 출력합니다.
