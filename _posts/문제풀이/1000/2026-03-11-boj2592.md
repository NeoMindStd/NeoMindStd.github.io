# 풀이
## 문제

10개의 수가 주어질 때
평균과 최빈값을 출력하는 문제입니다.

## 코드

```python
s=0
d={}
for _ in range(10):
    n=int(input())
    try:d[n]+=1
    except:d[n]=1
    s+=n
print(s//10)
print(max(d.items(),key=lambda x:x[1])[0])
```

## 설명

총합으로 평균을 계산하고,
딕셔너리 빈도수를 이용해
출현 횟수가 가장 많은 수를 찾습니다.
