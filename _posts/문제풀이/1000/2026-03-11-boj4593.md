# 풀이

## 문제

두 플레이어의 가위바위보 기록 문자열이 주어질 때,
각 게임에서의 승수를 집계해 출력하는 문제입니다.

## 코드

```python
while True:
    s1,s2=input(),input()
    p1=p2=0
    for i in range(len(s1)):
        a,b=s1[i],s2[i]
        if (a=='R' and b=='S') or\
           (a=='S' and b=='P') or\
           (a=='P' and b=='R'):
            p1+=1
        elif (b=='R' and a=='S') or\
             (b=='S' and a=='P') or\
             (b=='P' and a=='R'):
            p2+=1
    if s1 == 'E':
        break
    print(f'P1: {p1}\nP2: {p2}')
```

## 설명

같은 인덱스의 문자끼리 승패를 비교해 점수를 누적하고,
종료 문자열 `E`가 나오기 전까지 결과를 출력합니다.
