# 풀이

## 문제

N개의 줄에 걸쳐

"Hello World, Judge i!"

를 출력하는 프로그램을 만들라. 여기서 i는 줄의 번호이다.

## 코드

```python
for i in range(int(input())):print(f'Hello World, Judge {i+1}!')
```

## 설명

파이썬에서는 위와 같이 선언시 i는 0부터 시작하므로 i+1로 출력합니다.
