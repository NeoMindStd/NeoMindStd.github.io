# 풀이

## 문제

당신은 어떤 물건이라도 20% 할인해주는 쿠폰을 가지고 있다.

원래 가격이 주어질 때, 쿠폰을 사용하면 얼마가 되는지 알려주는 프로그램을 작성하시오.

**입력 요약**  
첫 번째 줄에 테스트케이스의 수가 주어진다.

각 줄에는 물건의 원래가격이 소수점 둘째자리까지 주어진다.

**출력 요약**  
할인된 가격을 달러 단위로 출력한다.

나누어떨어지지 않을 때는 소수점 셋째 자리에서 반올림해서 둘째 자리까지 출력한다.

## 코드

{% raw %}
```python
exec("print('$%.2f'%(float(input())*0.8));"*int(input()))
```
{% endraw %}

## 설명

핵심은 `구현` 관점에서 당신은 어떤 물건이라도 20% 할인해주는 쿠폰을 가지고 있다.

원래 가격이 주어질 때, 쿠폰을 사용하면 얼마가 되는지 알려주는 프로그램을 작성하시오.를 만족하도록 로직을 구성하는 것입니다.

코드는 입력을 파싱한 뒤 조건 분기와 계산을 순서대로 수행하고, 문제에서 요구한 형식으로 결과를 출력합니다.

경계값과 예외 케이스도 함께 고려해 오답이 나기 쉬운 상황을 방지합니다.
