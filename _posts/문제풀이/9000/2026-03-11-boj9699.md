# 풀이

## 문제

각 테스트케이스에서 주어진 수들 중 최댓값을 찾아 지정된 형식으로 출력하는 문제입니다.

## 코드

```python
for T in range(int(input())):print(f'Case #{T+1}: {max(map(int,input().split()))}')

```

## 설명

테스트케이스 번호를 함께 출력해야 하므로 반복문에서 `Case #x` 형식과 최댓값을 같이 출력합니다.
