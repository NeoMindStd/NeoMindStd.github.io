# 풀이

## 문제

0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

**입력 요약**  
첫째 줄에 정수 N(0 ≤ N ≤ 12)이 주어진다.

**출력 요약**  
첫째 줄에 N!을 출력한다.

## 코드

{% raw %}
```python
def factorial(n):
    if(n<=1): return 1
    else: return factorial(n-1)*n

print(factorial(int(input())))
```
{% endraw %}

## 설명

핵심은 `구현` 관점에서 0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.를 만족하도록 로직을 구성하는 것입니다.
