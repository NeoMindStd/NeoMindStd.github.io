# 풀이

## 문제

수하물 크기/무게 제한을 만족하는지 판별해
각 수하물의 가능 여부와 총 가능 개수를 출력하는 문제입니다.

## 코드

```python
s=[0]
for T in range(int(input())):
    a,b,c,d=map(float,input().split())
    if (a>56 or b>45 or c>25)and a+b+c>125 or d>7:s.append('0')
    else:
        s.append('1')
        s[0]+=1
print('\n'.join(s[1:]+[str(s[0])]))
```

## 설명

규격(개별 치수 또는 합계)과 무게 조건을 검사해
케이스마다 `1/0`을 출력하고 마지막에 통과 개수를 출력합니다.
