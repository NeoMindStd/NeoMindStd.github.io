# 풀이
## 문제

하이픈으로 구분된 단어 문자열에서
각 단어의 첫 글자(대문자)만 이어붙여 출력하는 문제입니다.

## 코드

```python
print("".join(map(str,filter(lambda c:c.isupper(),input()))))
```

## 설명

입력 문자열에서 대문자만 필터링해
바로 이어붙이면 약어가 됩니다.
