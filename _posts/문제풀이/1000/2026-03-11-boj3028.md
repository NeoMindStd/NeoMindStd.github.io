# 풀이

## 문제

컵 3개와 공 1개가 있을 때,
주어진 섞기 명령(`A`, `B`, `C`)을 순서대로 수행한 뒤 공의 위치를 출력하는 문제입니다.

## 코드

```python
l=list(input())
c=[True,False,False]
def A(c):c[0],c[1]=c[1],c[0]
def B(c):c[1],c[2]=c[2],c[1]
def C(c):c[0],c[2]=c[2],c[0]
for s in l:exec(f'{s}(c)')
print(c.index(True)+1)
```

## 설명

컵 상태를 불리언 배열로 관리하면서
명령마다 해당 두 컵의 값을 스왑하면 최종 위치를 구할 수 있습니다.
