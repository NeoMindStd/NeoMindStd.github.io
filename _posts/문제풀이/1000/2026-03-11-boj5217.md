# 풀이

## 문제

주어진 수 `n`을 서로 다른 두 양의 정수 합으로 나타내는
모든 쌍을 출력하는 문제입니다.

## 코드

```python
for T in range(int(input())):
    n,l=int(input()),[]
    for i in range(1,n//2+1):
        if 1<=n-i<=12 and i!=n-i:l.append(f'{i} {n-i}')
    print(f'Pairs for {n}:',', '.join(l))
```

## 설명

`i`를 절반까지만 순회하며 `(i, n-i)` 형태의 유효 쌍을 모아
지정된 형식으로 출력합니다.
