# 풀이
## 문제

100x100 도화지에 색종이를 여러 장 붙였을 때
검은 영역의 넓이를 구하는 문제입니다.

## 코드

```python
import sys
layer = [[0]*100 for _ in range(100)]
for _ in range(int(sys.stdin.readline())):
    r, c = map(int, sys.stdin.readline().split())
    for i in range(r, r+10):
        for j in range(c, c+10):
            layer[i][j] = 1
s = 0
for row in layer:
    s += sum(row)
print(s)
```

## 설명

도화지를 2차원 배열로 두고
색종이가 덮는 칸을 `1`로 표시한 뒤
전체 합을 계산하면 넓이가 됩니다.
