# 풀이
## 문제

점수들을 최댓값 기준으로 조정한 뒤
새 평균을 구하는 문제입니다.

## 코드

```python
a = int(input())
b = list(input().split())

for i in range(len(b)):
    b[i] = int(b[i])


bsum = 0
for i in range(len(b)):
    bsum += b[i]/max(b)*100

print(bsum/len(b))
```

## 설명

각 점수 `x`를 `(x / M) * 100`으로 바꿔 합산하고,
과목 수로 나누어 조정 평균을 계산합니다.

여기서 `M`은 원래 점수 중 최댓값입니다.
