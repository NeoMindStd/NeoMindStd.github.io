# 풀이
## 문제

8x8 체스판 상태가 주어질 때
하얀 칸에 있는 `F` 말의 개수를 세는 문제입니다.

## 코드

```python
lane = list()
for i in range(8):
    lane.append(input())

white = 1
count = 0

for i in lane:
    for j in i:
        if(j=='F' and white==1):
            count += 1
        white = (white+1)%2
    white = (white+1)%2
print(count)
```

## 설명

좌상단이 하얀 칸이므로
칸 색을 번갈아가며 추적하면서 `F`를 카운트합니다.

각 행이 끝날 때 시작 색이 반전되도록 한 번 더 토글하면
체스판의 색 패턴을 그대로 구현할 수 있습니다.
