# 풀이

## 문제

다섯 개 숫자를 버블 정렬처럼 교환하면서,
교환이 일어날 때마다 현재 배열 상태를 출력하는 문제입니다.

## 코드

```python
woods = list(map(int, input().split()))

for i in range(len(woods)-1, 0, -1):
    for j in range(i):
        if woods[j] > woods[j+1]:
            woods[j], woods[j+1] = woods[j+1], woods[j]
            print(*woods)
```

## 설명

인접한 두 수를 비교해 순서가 뒤집혀 있으면 교환하고,
교환 직후 배열을 출력하면 문제 조건과 동일하게 동작합니다.
