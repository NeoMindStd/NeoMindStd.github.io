# 풀이

## 문제

등차수열 첫항 `a`, 공차 `b`, 목표값 `c`가 주어질 때
`c`가 수열 몇 번째 항인지 출력하고, 없으면 `X`를 출력하는 문제입니다.

## 코드

```python
import sys
read=sys.stdin.readline
while True:
    a,b,c=map(int,read().split())
    if a==b==c==0:
        break
    else:
        t=abs((c-a)//b)
        if (c-a)%b != 0 or a+t*b!=c: print('X')
        else: print(t+1)
```

## 설명

`c-a`가 공차 `b`로 나누어떨어지는지 확인하고,
정수 항 번호가 성립하면 `t+1`을, 아니면 `X`를 출력합니다.
