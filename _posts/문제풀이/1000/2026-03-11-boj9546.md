# 풀이

## 문제

버스 정류장을 지날 때마다
승객 수가 `절반 내리고 +1` 규칙을 거쳤다고 할 때,
처음 승객 수를 구하는 문제입니다.

## 코드

```python
for T in range(int(input())):
    k,s=int(input()),0
    for i in range(k):
        s+=0.5
        s*=2
    print(int(s))
```

## 설명

역순으로 생각하면 이전 정류장 승객 수는 `(현재+0.5)*2`가 되어,
해당 연산을 `k`번 반복해 시작 승객 수를 구합니다.
