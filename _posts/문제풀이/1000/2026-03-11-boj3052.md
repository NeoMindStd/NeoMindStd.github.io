# 풀이

## 문제

10개 수를 42로 나눈 나머지들 중
서로 다른 값의 개수를 구하는 문제입니다.

## 코드

```python
print(len(list(set([int(input())%42 for _ in range(10)]))))
```

## 설명

나머지 값을 `set`에 넣으면 중복이 제거되므로
집합 크기가 곧 정답입니다.
