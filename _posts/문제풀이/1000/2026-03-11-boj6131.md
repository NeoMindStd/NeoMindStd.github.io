# 풀이

## 문제

`a^2 - b^2 = n`을 만족하는
자연수 쌍 `(a, b)`의 개수를 구하는 문제입니다.

## 코드

```python
n=int(input())
c=0
for a in range(1+n,100001):
    b=a-n
    if int(a**.5)**2==a and int(b**.5)**2==b:c+=1
print(c)
```

## 설명

`a^2 - b^2 = n`을 이용해 가능한 후보를 순회하면서
두 값이 모두 완전제곱수인지 검사해 개수를 셉니다.
