# 풀이

## 문제

의사가 요구한 `"a"` 길이와 환자가 낼 수 있는 `"a"` 길이를 비교해
가능하면 `go`, 아니면 `no`를 출력하는 문제입니다.

## 코드

```python
a, b = input(), input()
print("go" if len(a) >= len(b) else "no")
```

## 설명

문자열 길이 비교만으로 판정할 수 있습니다.
