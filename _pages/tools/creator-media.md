# 방송·오디오·영상 도구 허브

녹화, 스트리밍, 오디오 편집, 작곡, 영상 편집처럼 크리에이터 작업에 가까운 도구만 모았습니다.

### 작업 흐름별 빠른 선택표

| 목적 | 먼저 볼 글 | 잘 맞는 상황 | 다음 단계 |
|:--|:--|:--|:--|
| 화면 녹화와 방송 송출을 시작할 때 | [OBS 설치와 사용법](/tools/obs-studio-guide/) | 무료로 녹화와 방송을 동시에 시작하고 싶을 때 | XSplit, DaVinci Resolve |
| 방송 장면 구성과 송출 UI를 더 쉽게 다루고 싶을 때 | [XSplit 설치와 사용법](/tools/xsplit-broadcaster-guide/) | GUI 중심 방송 환경을 선호할 때 | OBS |
| 빠른 오디오 편집이 필요할 때 | [GoldWave 설치와 사용법](/tools/goldwave-guide/) | 컷 편집, 잡음 제거, 간단한 오디오 작업이 많을 때 | Cakewalk |
| 음악 제작과 DAW가 필요할 때 | [Cakewalk 설치와 사용법](/tools/cakewalk-guide/) | 멀티트랙 작업과 MIDI를 다뤄야 할 때 | GoldWave |
| 본격적인 영상 편집과 렌더가 필요할 때 | [DaVinci Resolve 설치와 사용법](/tools/davinci-resolve-guide/) | 컷 편집, 컬러, 렌더까지 한 툴에서 하고 싶을 때 | OBS |

### 추천 순서

1. 녹화나 송출이 먼저면 `OBS`를 시작점으로 잡는 편이 가장 무난합니다.
2. 오디오만 빠르게 손보고 싶다면 `GoldWave`, 음악 제작까지 갈 거면 `Cakewalk`로 이어서 보면 됩니다.
3. 영상 편집까지 필요한 경우엔 마지막에 `DaVinci Resolve`로 넘어가면 흐름이 잘 이어집니다.
