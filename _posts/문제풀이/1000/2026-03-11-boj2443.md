# 풀이
## 문제

입력 `N`에 대해
첫 줄이 가장 넓고 아래로 갈수록 줄어드는 역피라미드 형태의 별을 출력하는 문제입니다.

## 코드

```python
a = int(input())

for i in range(a):
    for j in range(i):
        print(" ", end="")
    for j in range(i, 2*a-i-1):
        print("*", end="")
    print()
```

## 설명

`i`번째 줄에서
앞 공백은 `i`개,
별은 `2*(N-i)-1`개를 출력하면 됩니다.
