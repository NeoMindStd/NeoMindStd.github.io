# 풀이

## 문제

정수 `n`이 주어질 때
식 `n^2 - n + 1` 값을 출력하는 문제입니다.

## 코드

```python
while True:
    n = int(input())
    if n == 0: break

    print(f'{n} => {n * n - n + 1}')
```

## 설명

입력이 `0`이 나오기 전까지 반복하며
주어진 식을 계산해 형식에 맞춰 출력합니다.
