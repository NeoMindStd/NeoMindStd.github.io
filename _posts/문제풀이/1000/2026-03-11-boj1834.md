# 풀이
## 문제

정수 `N`이 주어질 때
`N`으로 나눈 몫과 나머지가 같은 양의 정수들을 모두 더하는 문제입니다.

## 코드

```python
n=int(input())
s = 0
for i in range(n): s += i*n+i
print(s)
```

## 설명

몫과 나머지를 `i`라 두면 수는 `N*i + i` 형태입니다.
`i=0`부터 `N-1`까지 합산하면 정답이 됩니다.
