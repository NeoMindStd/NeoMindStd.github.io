# 풀이
## 문제

주어진 수 `N`을 이용해
특정 규칙의 합을 계산하는 문제입니다.

코드는 `N`의 이진 표현을 사용해
`3`의 거듭제곱 합으로 변환합니다.

## 코드

```python
n=str(bin(int(input())))[:1:-1]

answer = 0
for i in range(len(n)):
    if n[i]=='1': answer += 3**i

print(answer)
```

## 설명

`N`의 이진수에서 1인 비트 위치 `i`마다
`3^i`를 더합니다.

즉, 2진수 자릿값을 3진 거듭제곱 합으로 대응해 계산합니다.
