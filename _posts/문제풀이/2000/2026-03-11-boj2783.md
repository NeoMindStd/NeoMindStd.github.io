# 풀이

## 문제

기준 중량(1000g)당 가격을 계산해 가장 저렴한 삼각 김밥의 가격을 구하는 문제입니다.

## 코드

```python
x,y=map(int,input().split())
s=1000/y*x
for i in range(int(input())):
    x,y=map(int,input().split())
    s=min(1000/y*x,s)
print(s)
```

## 설명

초기 제품과 비교 제품들을 모두 `1000 / 중량 * 가격`으로 환산해 최솟값을 출력합니다.
