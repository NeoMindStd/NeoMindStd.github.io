# 풀이

## 문제

거리와 두 속도가 주어질 때
소요 시간 차이를 `h:mm:ss` 형식으로 출력하는 문제입니다.
입력 종료 조건이 있습니다.

## 코드

```python
while True:
    a,b,c=map(int,input().split())
    if b<1:break
    t=a*3600/b-a*3600/c
    h=t//3600
    t%=3600
    m=t//60
    print('%d:%02d:%02d'%(h,m,round(t%60)))
```

## 설명

두 경우의 시간을 초 단위로 계산해 차를 구한 뒤,
시/분/초로 변환해 형식에 맞춰 출력합니다.
