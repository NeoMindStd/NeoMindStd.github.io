# 풀이

## 문제

주어진 ASCII 그림을 그대로 출력하는 문제입니다.

## 코드

```python
print("""     /~\\
    ( oo|
    _\=/_
   /  _  \\
  //|/.\\|\\\\ 
 ||  \\ /  ||
============
|          |
|          |
|          |""")
```

## 설명

백슬래시 이스케이프를 포함해 문자열을 그대로 출력하면 됩니다.
