# 풀이

## 문제

기준 수 `n`이 주어질 때,
입력되는 수가 `n`의 배수인지 판별해 형식에 맞게 출력하는 문제입니다.

## 코드

```python
n=int(input())
while True:
    a=int(input())
    if a==0: break
    print('%d is%sa multiple of %d.' %(a, ' NOT 'if a%n!=0 else' ', n))
```

## 설명

`a % n == 0` 여부를 검사해
문제에서 요구한 문장 형태로 출력하면 됩니다.
