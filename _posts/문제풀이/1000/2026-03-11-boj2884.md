# 풀이
## 문제

현재 시각에서 45분 앞선 시간을 출력하는 문제입니다.

## 코드

```python
h, m = map(int, input().split())
tm = 60 * (h+24) + m - 45
print((tm // 60)%24, tm % 60)
```

## 설명

시각을 분 단위로 변환해 45를 빼고
다시 시/분으로 환산하면 경계 시간도 안전하게 처리됩니다.
