# 풀이
## 문제

집합 `A`와 질의 수열 `B`가 주어질 때,
각 `B` 원소가 `A`에 존재하는지 출력하는 문제입니다.

## 코드

```python
_ = int(input())
a = set(map(int, input().split()))
_ = int(input())
b = list(map(int, input().split()))

for each_b in b:
    print(1 if each_b in a else 0)
```

## 설명

`A`를 `set`으로 만들어 membership 조회를 `O(1)` 평균 시간에 처리합니다.

질의마다 포함 여부를 `1/0`으로 출력합니다.
