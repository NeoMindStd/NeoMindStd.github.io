# 풀이

## 문제

세 캥거루의 위치가 주어질 때,
규칙에 맞게 이동해서 만들 수 있는 최대 이동 횟수를 구하는 문제입니다.

## 코드

```python
a, b, c = map(int, input().split())
print(max(b-a, c-b)-1)
```

## 설명

두 간격 중 더 큰 쪽에서 한 칸씩 채워 넣는 것이 최대 이동 횟수이므로
`max(b-a, c-b) - 1`이 정답입니다.
