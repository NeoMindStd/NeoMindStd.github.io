# 풀이

## 문제

두 정수 n과 m이 주어졌을 때, 0 < a < b < n인 정수 쌍 (a, b) 중에서 (a2+b2+m)/(ab)가 정수인 쌍의 개수를 구하는 프로그램을 작성하시오.

**입력 요약**  
첫째 줄에 테스트 케이스의 개수 T가 주어진다. 각 테스트 케이스는 한 줄로 이루어져 있으며, n과 m이 주어진다. 두 수는 0보다 크고, 100보다 작거나 같다.

**출력 요약**  
각 테스트 케이스마다 문제의 조건을 만족하는 (a, b)쌍의 개수를 출력한다.

## 코드

{% raw %}
```python
for T in range(int(input())):
    n, m = map(int, input().split())
    cnt = 0
    for b in range(2, n):
        for a in range(1, b):
            if (a**2 + b**2 + m) / (a*b) == (a**2 + b**2 + m) // (a*b):
                cnt += 1
    print(cnt)
```
{% endraw %}

## 설명

핵심은 `구현` 관점에서 두 정수 n과 m이 주어졌을 때, 0 < a < b < n인 정수 쌍 (a, b) 중에서 (a2+b2+m)/(ab)가 정수인 쌍의 개수를 구하는 프로그램을 작성하시오.를 만족하도록 로직을 구성하는 것입니다.
