# 풀이
## 문제

5개의 문자열 중
`FBI`를 포함하는 문자열의 번호를 모두 출력하는 문제입니다.
없으면 `HE GOT AWAY!`를 출력합니다.

## 코드

```python
l=[input()for _ in range(5)]
s=''
for i in range(5):
    if l[i].count('FBI')>0:s+=f'{i+1} '
print(s if s!='' else'HE GOT AWAY!')
```

## 설명

각 문자열에 `FBI`가 포함되는지 검사하고
해당 인덱스(1-based)를 누적 출력합니다.
