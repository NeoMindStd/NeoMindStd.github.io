# 풀이
## 문제

문장을 한 줄씩 입력받아
각 줄에 포함된 모음(`a,e,i,o,u`) 개수를 출력하는 문제입니다.

입력이 `#`이면 종료합니다.

## 코드

```python
while True:
    s=input().lower()
    if s=='#':break
    t=0
    for c in s:
        if c in ['a','e','i','o','u']: t+=1
    print(t)
```

## 설명

대소문자 구분을 없애기 위해 문자열을 `lower()`로 바꾼 뒤,
문자 하나씩 순회하며 모음 여부를 검사해 카운트합니다.
