# 풀이
## 문제

오른쪽 정렬된 별이
점점 늘었다가 다시 줄어드는 모양을 출력하는 문제입니다.

## 코드

```python
n = int(input())

for i in range(n):
    for j in range(n-1-i):
        print(' ', end='')
    for j in range(i+1):
        print('*', end='')
    print()
    
for i in range(1, n):
    for j in range(i):
        print(' ', end='')
    for j in range(n-i):
        print('*', end='')
    print()
```

## 설명

윗부분은 공백 감소/별 증가,
아랫부분은 공백 증가/별 감소 형태로 두 구간을 출력하면 됩니다.
