# 중기계획 2차 실행: 허브 비교표와 추천 순서 추가

## 목적

- 허브 페이지를 단순 링크 모음이 아니라 `선택 도구`처럼 보이게 만들기
- 독자가 "무엇부터 읽어야 할지" 빠르게 결정할 수 있게 비교표와 추천 순서를 제공하기

## 반영한 허브

### AI 코딩 도구 허브

- 도구별 `잘 맞는 사람 / 인터페이스 / 추천 시작점` 비교표 추가
- `CLI 먼저`, `IDE 먼저`, `개념 먼저`로 나눈 추천 순서 추가

### AI 브라우저·리서치 허브

- `조사형 / 브라우저 내장형 / 작업형 브라우저 / 심화 리서치` 비교표 추가
- 브라우저 변경 의향과 사용 습관 기준 추천 순서 추가

### 로컬 LLM·셀프호스팅 허브

- `UI / 환경 / 메모리·장비 감각` 기준 비교표 추가
- `입문 -> UI/CLI 선택 -> 셀프호스팅 확장` 순서 추가

## 기대 효과

1. 허브 자체에서 이탈하지 않고 다음 클릭까지 이어질 가능성 증가
2. Search Console snippet에서 허브 요약 가치가 더 분명해질 가능성
3. 앞으로 클러스터별 신규 글을 추가할 때 배치 기준이 더 명확해짐

## 다음 후보

1. 실무용 AI·워크플로우 허브에도 `업무형 / 리서치형 / 에이전트형` 비교표 추가
2. 각 허브 대표 영역에 썸네일 카드나 시각적 배지 추가
3. Search Console이 잡히는 허브부터 title/description 2차 조정
