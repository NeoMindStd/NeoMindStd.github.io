# 풀이

## 문제

입력된 이름들을 모두 소문자로 변환해 출력하는 문제입니다.

## 코드

```python
exec("print(input().lower());"*int(input()))
```

## 설명

첫 줄의 개수만큼 문자열을 입력받아
`lower()`를 적용해 한 줄씩 출력합니다.
