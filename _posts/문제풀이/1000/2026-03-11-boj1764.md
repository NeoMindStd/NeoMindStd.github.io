# 풀이
## 문제

듣지 못한 사람 목록과 보지 못한 사람 목록이 주어질 때
두 목록에 모두 포함된 사람을 사전순으로 출력하는 문제입니다.

## 코드

```python
import sys

n, m = map(int, sys.stdin.readline().split())
d = set()
b = set()
mans = []

for _ in range(n):
    mans.append(sys.stdin.readline())
    d.add(mans[-1])
    
for _ in range(m):
    mans.append(sys.stdin.readline())
    b.add(mans[-1])


dbj = []
mans = list(set(mans))
for man in mans:
    if man in d and man in b:
        dbj.append(man)
dbj.sort()

print(len(dbj))
print("".join(map(str, dbj)))
```

## 설명

두 집합(`d`, `b`)을 만든 뒤
교집합에 해당하는 이름만 모아 정렬합니다.

인원 수와 이름 목록을 출력합니다.
