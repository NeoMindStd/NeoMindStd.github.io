# 풀이

## 문제

2진수 문자열이 주어질 때
17배 한 결과를 2진수로 출력하는 문제입니다.

## 코드

```python
print(format(int(input(),2)*17,'b'))
```

## 설명

입력을 정수로 변환한 뒤 `17`을 곱하고,
`format(..., 'b')`로 2진수 문자열로 출력합니다.
