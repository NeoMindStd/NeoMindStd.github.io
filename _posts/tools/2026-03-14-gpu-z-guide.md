# 무엇인가

GPU-Z는 그래픽카드 모델, BIOS, PCIe 링크 상태, VRAM 종류, 센서 수치를 빠르게 확인하는 데 특화된 도구입니다. 드라이버 문제를 잡거나 게임 중 클럭 변화를 보려 할 때 특히 자주 쓰입니다.

- 다운로드: [공식 다운로드 페이지](https://www.techpowerup.com/gpuz/)

# 지원 OS와 설치 방법

## Windows

공식 다운로드 페이지에서 Windows용 설치 파일이나 패키지를 내려받아 설치합니다. 설치가 끝나면 첫 실행과 버전 확인을 한 번 진행하는 편이 좋습니다. 포터블처럼 바로 실행하는 경우도 많습니다.

## macOS

현재 공식 macOS 배포를 확인하기 어렵습니다. 공식 macOS 버전은 없습니다.

## Linux

현재 공식 Linux 배포를 확인하기 어렵습니다. 공식 Linux 버전은 없습니다.

# 기본 사용법

1. Graphics Card 탭에서 칩셋과 VRAM, 버스 인터페이스를 확인합니다.
2. Sensors 탭으로 이동해 온도, 전력, 팬 속도 같은 실시간 값을 읽습니다.
3. 부하 중 로깅을 켜서 병목이 어디서 생기는지 확인합니다.

# 주의할 점

GPU-Z는 정보 확인에는 매우 좋지만, 숫자 하나만 보고 성능을 단정하면 오해하기 쉽습니다. 센서 값은 실제 부하 상황과 함께 봐야 의미가 있습니다.

# 요약

GPU-Z는 그래픽카드 정보를 가장 빠르게 읽는 도구 중 하나입니다. 설치는 간단하고, 핵심은 Sensors 탭과 로깅 기능을 실제 부하 상황과 함께 보는 것입니다.
