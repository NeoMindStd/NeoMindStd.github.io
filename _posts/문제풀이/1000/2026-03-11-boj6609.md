# 풀이

## 문제

모기 개체수 모델의 초기값과 계수를 입력받아
`n`세대 후 성충 수를 구하는 문제입니다.

## 코드

```python
import sys
for line in sys.stdin:
    m,p,l,e,r,s,n=map(int,line.split())
    for i in range(n):
        m,p,l=p//s,l//r,e*m
    print(m)
```

## 설명

각 세대마다 문제의 전이식대로 `m`, `p`, `l`을 갱신하고
반복이 끝난 뒤 성충 `m` 값을 출력합니다.
