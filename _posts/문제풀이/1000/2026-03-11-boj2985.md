# 풀이

## 문제

세 수 `a, b, c`가 주어질 때
사칙연산을 이용해 성립하는 식을 한 줄 출력하는 문제입니다.

## 코드

```python
a, b, c = map(int, input().split())
ops = '+-*/'
answer = ''
for op in ops:
    if eval(f'{a}=={b}{op}{c}'): answer = f'{a}={b}{op}{c}'
    if eval(f'{a}{op}{b}=={c}'): answer = f'{a}{op}{b}={c}'
print(answer)
```

## 설명

가능한 네 연산자를 순회하며
`a = b op c` 또는 `a op b = c` 형태가 성립하는 식을 찾아 출력합니다.
