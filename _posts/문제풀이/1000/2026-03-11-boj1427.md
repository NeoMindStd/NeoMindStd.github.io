# 풀이
## 문제

숫자 `N`의 각 자릿수를 내림차순으로 정렬해 출력하는 문제입니다.

## 코드

```python
n = list(input())
n.sort(reverse=True)
for i in n:
    print(i, end="")
```

## 설명

입력을 문자열 리스트로 받아
`reverse=True` 정렬 후 이어서 출력하면 됩니다.
