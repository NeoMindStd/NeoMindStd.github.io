# 풀이

## 문제

두 수를 더할 때 발생하는 받아올림(carry)의 횟수를 구하는 문제입니다.

## 코드

```python
while True:
    a,b=map(list,input().split())
    a=list(map(int,a))
    b=list(map(int,b))
    if a[0]==b[0]==0:break
    a.reverse()
    b.reverse()
    c=0
    for i in range(len(a), max(len(a),len(b))+1):a.append(0)
    for i in range(len(b), len(a)):b.append(0)
    for i in range(len(a)-1):
        if a[i]+b[i]>9:
            a[i+1]+=1
            c+=1
    print(c)
```

## 설명

자릿수를 뒤집어 낮은 자리부터 더하면서
합이 10 이상이면 다음 자리에 1을 넘기고 카운트를 증가시켜 횟수를 계산합니다.
