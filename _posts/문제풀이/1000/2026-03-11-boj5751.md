# 풀이

## 문제

동전 결과에서 `1`의 개수와 `0`의 개수를 세어
John과 Mary의 승수를 출력하는 문제입니다.

## 코드

```python
while True:
    n=int(input())
    if n==0:break
    x=sum(map(int,input().split()))
    print(f'Mary won {n-x} times and John won {x} times')
```

## 설명

`1`의 합을 John 승수로 보고,
전체 횟수 `n`에서 빼서 Mary 승수를 계산합니다.
