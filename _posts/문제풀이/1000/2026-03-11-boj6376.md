# 풀이

## 문제

자연상수 `e`의 부분합을
지정된 출력 형식으로 표로 출력하는 문제입니다.

## 코드

```python
a=[1,1]
while a[-1]<=10**9:
    a.append(a[-1]*len(a))
r=['n e','- -----------']
for i in range(2):
    s='%d %.0f' %(i, sum(map(lambda x: 1/x, a[:i+1])))
    r.append(s)
for i in range(2,3):
    s='%d %.1f' %(i, sum(map(lambda x: 1/x, a[:i+1])))
    r.append(s)
for i in range(3,10):
    s='%d %.9f' %(i, sum(map(lambda x: 1/x, a[:i+1])))
    r.append(s)
print('\n'.join(r))
```

## 설명

`e = Σ(1/n!)` 부분합을 계산하고,
문제에서 요구한 자리수 포맷에 맞춰 `n = 0..9` 행을 출력합니다.
