# 풀이

## 문제

두 기종 비행기의 좌석 수가 주어질 때
총 좌석 수를 계산하는 문제입니다.

## 코드

```python
a,b,c,d=map(int,input().split())
print(a*b+c*d)
```

## 설명

기종별 좌석 수(`행 * 열`)를 구해
합계를 출력하면 됩니다.
