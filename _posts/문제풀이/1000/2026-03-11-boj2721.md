# 풀이
## 문제

각 테스트케이스 `n`에 대해
문제에서 정의한 `W(n)` 값을 계산해 출력하는 문제입니다.

## 코드

```python
l=[k*(k+1)//2 for k in range(302)]
for T in range(int(input())):
    n=int(input())
    w=0
    for i in range(1,n+1):w+=i*l[i+1]
    print(w)
```

## 설명

삼각수 배열을 미리 만들고,
공식 `W(n)=Σ(i*T(i+1))`를 반복문으로 계산합니다.
