# 풀이
## 문제

다섯 수를 제곱해 더한 뒤,
합을 10으로 나눈 나머지를 출력하는 문제입니다.

## 코드

```python
a = list(map(int, input().split()))
for i in range(5):
    a[i] **= 2
print(sum(a)%10)
```

## 설명

입력값 5개를 제곱 처리한 뒤
합계의 `mod 10` 결과를 그대로 출력하면 됩니다.
