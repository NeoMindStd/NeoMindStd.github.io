# 풀이
## 문제

세 줄의 숫자 문자열 각각에 대해
같은 숫자가 연속으로 가장 길게 이어진 길이를 구하는 문제입니다.

## 코드

```python
for T in range(3):
    t,a,p='',0,1
    for c in input():
        if t==c: p+=1
        else: p=1
        a=max(a,p)
        t=c
    print(a)
```

## 설명

직전 문자와 현재 문자를 비교해
연속 길이 `p`를 갱신하고,
최대 길이 `a`를 유지하면 됩니다.
