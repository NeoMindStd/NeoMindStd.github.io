# 풀이

## 문제

세 변 길이로 만들 수 있는 삼각형을
`invalid / equilateral / isosceles / scalene`으로 분류하는 문제입니다.

## 코드

```python
for T in range(int(input())):
    l=sorted(map(int,input().split()))
    if l[2]>=l[0]+l[1]:r='invalid!'
    elif l[2]==l[0]:r='equilateral'
    elif l[2]==l[1]or l[1]==l[0]:r='isosceles'
    else:r='scalene'
    print(f'Case #{T+1}: {r}')
```

## 설명

정렬 후 삼각형 성립 조건을 먼저 검사하고,
변의 동일성에 따라 정삼각형/이등변/부등변으로 구분합니다.
