# 풀이

## 문제

연속 입력되는 값들의 차이를 소수 둘째 자리까지 출력하는 문제입니다.
입력 `999`가 나오면 종료합니다.

## 코드

```python
import sys;read=sys.stdin.readline
l=[]
while True:
    t=float(read())
    if t==999:break
    else:l.append(t)
r=[]
for i in range(len(l)-1):r.append('%.2f' %(l[i+1]-l[i]))
print('\n'.join(r))
```

## 설명

값들을 배열에 저장한 뒤
인접한 두 값의 차를 계산해 형식에 맞게 출력합니다.
