# 풀이

## 문제

정수 `n`과 뒤집은 수를 더한 결과가
회문수인지 판별해 `YES`/`NO`를 출력하는 문제입니다.

## 코드

```python
for T in range(int(input())):
    n = int(input())
    m = int(str(n)[::-1])
    t = str(n + m)
    print('YES' if t == t[::-1] else 'NO')
```

## 설명

문자열 뒤집기를 이용해 역수를 만들고,
합 결과 문자열이 앞뒤가 같은지 검사하면 됩니다.
