# 풀이

## 문제

정사각형 격자에서 만들 수 있는 모든 정사각형 개수를
입력값마다 구하는 문제입니다.

## 코드

```python
while True:
    n,s=int(input()),0
    if n==0:break
    for i in range(n,0,-1):s+=i*i
    print(s)
```

## 설명

`1^2 + 2^2 + ... + n^2`를 계산하면
전체 정사각형 개수가 됩니다.
