# 풀이
## 문제

두 이진수 문자열 `A`, `B`가 주어질 때
합을 이진수 형태로 출력하는 문제입니다.

## 코드

```python
a, b = input().split()
print(format(int(a, 2)+int(b, 2), 'b'))
```

## 설명

`int(x, 2)`로 이진수를 정수로 변환해 더한 뒤,
`format(sum, 'b')`로 다시 이진 문자열로 바꿔 출력합니다.
