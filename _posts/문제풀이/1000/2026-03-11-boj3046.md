# 풀이

## 문제

두 수의 평균 `S`와 첫 번째 수 `R1`이 주어질 때
두 번째 수 `R2`를 구하는 문제입니다.

## 코드

```python
r1, s = map(int, input().split())
print(s*2 - r1)
```

## 설명

평균 공식 `S = (R1 + R2) / 2`를 변형하면
`R2 = 2S - R1`입니다.
