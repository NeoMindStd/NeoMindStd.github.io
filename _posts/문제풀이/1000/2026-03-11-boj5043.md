# 풀이

## 문제

`b`비트로 표현 가능한 수의 개수 한도 내에서
`n`개를 모두 표현할 수 있는지 판별하는 문제입니다.

## 코드

```python
n,b=map(int,input().split())
print('yes'if n<=2**(b+1)-1 else'no')
```

## 설명

문제 조건식에 맞는 최대 개수 `2^(b+1)-1`과 `n`을 비교해
가능 여부를 출력합니다.
