# 풀이

## 문제

5장의 카드 점수를 입력받아 총합을 출력하는 문제입니다.

## 코드

```python
print(sum([int(input()) for _ in range(5)]))
```

## 설명

리스트 컴프리헨션으로 입력을 모아
`sum`으로 합산하면 됩니다.
