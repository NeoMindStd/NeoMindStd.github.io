# 풀이
## 문제

평균값(올림 기준) `I`와 곡 개수 `A`가 주어질 때
가능한 최소 총 멜로디 수를 구하는 문제입니다.

## 코드

```python
a, i = map(int, input().split())
print((i-1)*a +1)
```

## 설명

조건 `S/A < I`를 만족하는 최소 정수 `S`는
`(I-1)*A + 1`입니다.
