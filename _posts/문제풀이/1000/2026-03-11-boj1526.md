# 풀이
## 문제

주어진 수 `N` 이하의 금민수(4와 7로만 이루어진 수) 중
가장 큰 값을 찾는 문제입니다.

## 코드

```python
for i in range(int(input()), 3, -1):
    s = str(i)
    if s.count('4') + s.count('7') == len(s):
        print(i)
        break
```

## 설명

`N`부터 4까지 역순으로 내려오면서
해당 수의 각 자리가 4 또는 7인지 검사합니다.

처음 조건을 만족한 수가 곧 정답입니다.
