# 풀이
## 문제

각 테스트케이스마다
문자열의 각 문자를 `R`번 반복한 결과를 출력하는 문제입니다.

## 코드

```python
n = int(input())

for i in range(n):
    k, s = input().split()
    for ch in s:
        for j in range(int(k)):
            print(ch, end="")
    print()
```

## 설명

문자열을 순회하면서
각 문자를 반복 횟수만큼 출력하면 됩니다.
