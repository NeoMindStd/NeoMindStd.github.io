# 풀이
## 문제

규칙에 따라 점이 추가되는 도형에서
`N`번째 줄의 점 개수를 `45678`로 나눈 값을 구하는 문제입니다.

## 코드

```python
n = int(input())
print(((3*n*n+5*n+2)//2)%45678)
```

## 설명

문제의 점화식을 정리하면
`N`번째 값은 `((3n^2 + 5n + 2) / 2)` 형태로 계산됩니다.

요구 조건에 맞춰 마지막에 `45678`로 나눈 나머지를 출력합니다.
