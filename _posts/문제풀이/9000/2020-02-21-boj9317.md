# 풀이

## 문제

`W` = 화면 가로 길이, `H` = 화면 세로 길이, `D` = 화면 대각선 길이라고 정의합니다.

모니터의 비율은 가로:세로 = 16:9 입니다.

D와 가로 해상도, 세로 해상도가 주어졌을 때 수평 DPI와 수직 DPI를 구하시오.



## 코드

```python
while True:
    d,rh,rv=map(float,input().split())
    if d+rh+rv==0:break
    w=16/337**.5*d
    h=w*9/16
    print('Horizontal DPI: %.2f'%(rh/w))
    print('Vertical DPI: %.2f'%(rv/h))
```
