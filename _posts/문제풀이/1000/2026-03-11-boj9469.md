# 풀이

## 문제

여러 테스트케이스에서
주어진 값으로 계산한 결과를 지정된 포맷으로 출력하는 문제입니다.

## 코드

```python
exec("n,d,a,b,f=map(float,input().split());print('%d %.6f' %(n,d/(a+b)*f));"*int(input()))
```

## 설명

각 케이스마다 식 `d/(a+b)*f`를 계산하고,
케이스 번호와 함께 소수점 6자리까지 출력합니다.
