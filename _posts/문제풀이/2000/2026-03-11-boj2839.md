# 풀이

## 문제

3kg, 5kg 봉지를 사용해 정확히 Nkg를 만들 때 최소 봉지 수를 구하는 문제입니다.

## 코드

```python
a = int(input())

aDiv5 = a // 5

flag = False

for i in range(aDiv5, -1, -1):
    ami = a-(i*5)
    amiDiv3 = ami // 3

    for j in range(amiDiv3, -1, -1):
        if(i*5 + j*3 == a):
            print(i+j)
            flag = True
            break
        
    if(flag):
        break
        
if(not flag):
    print(-1)
```

## 설명

5kg 봉지 개수를 큰 값부터 줄여가며 가능한 3kg 조합을 찾고, 최초 해의 봉지 수를 출력합니다.
