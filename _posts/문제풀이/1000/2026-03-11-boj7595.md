# 풀이

## 문제

입력된 `n`에 대해
`*`로 높이 `n`인 직각삼각형을 출력하는 문제입니다.
`0`이 들어오면 종료합니다.

## 코드

```python
while True:
    n=int(input())
    if n==0:break
    for i in range(1,n+1):print('*'*i)
```

## 설명

반복 입력을 받다가 `0`이면 종료하고,
그 외에는 `1`부터 `n`까지 `*` 개수를 늘려가며 출력합니다.
