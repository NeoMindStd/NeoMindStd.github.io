# 풀이

## 문제

우주선 성능 정보와 거리 `d`가 주어질 때
`d` 거리 내 이동 가능한 우주선 수를 세는 문제입니다.

## 코드

```python
for T in range(int(input())):
    n,d=map(int,input().split())
    l=[list(map(int,input().split()))for _ in range(n)]
    s=0
    for i in l:
       if d/i[0]*i[2]<=i[1]:s+=1
    print(s)
```

## 설명

각 우주선에 대해 `필요 시간 = d/v`, `사용 가능 시간 = f/c`를 비교해
도달 가능하면 카운트를 증가시킵니다.
