# 풀이

## 문제

반대 방향으로 이동하는 두 개미 그룹이
`t`초 후 어떤 순서가 되는지 구하는 문제입니다.

## 코드

```python
cnt = 0
def ant_mapping(ant):
    global cnt
    cnt += 1
    return (cnt, ant)

n1, n2 = map(int, input().split())
ant1 = list(map(ant_mapping, input()[::-1]))
ant2 = input()
t = int(input())
cnt -= t + .5
ant2 = list(map(ant_mapping, ant2))
print(''.join(list(map(lambda x:x[1], sorted(ant1+ant2)))))
```

## 설명

첫 번째 그룹은 뒤집어 배치한 뒤,
시간 `t`를 반영한 정렬 키를 부여해 두 그룹을 합쳐 정렬하면
최종 순서를 얻을 수 있습니다.
