# 풀이

## 문제

남자 친구 수와 여자 친구 수를 입력받아
총 친구 수를 출력하는 문제입니다.

## 코드

```python
while True:
    a,b=map(int,input().split())
    if a==b==0:break
    print(a+b)
```

## 설명

입력이 `0 0`일 때 종료하고,
그 외에는 두 수의 합을 출력하면 됩니다.
