# 풀이
## 문제

세 주사위의 눈 개수 범위가 각각 `S1`, `S2`, `S3`일 때,
합이 가장 많이 나오는 값을 구하는 문제입니다.

## 코드

```python
a,b,c=map(int,input().split())
l={}
for i in range(a):
    for j in range(b):
        for k in range(c):
            s=i+j+k+3
            try:l[s]+=1
            except:l[s]=1
print(max(l.items(), key=lambda x:x[1])[0])
```

## 설명

세 주사위의 모든 조합을 순회해
합 `s`의 등장 횟수를 딕셔너리에 누적합니다.

마지막에 가장 많이 등장한 합의 값을 출력하면 됩니다.
