# 풀이

## 문제

좌표 점들을 입력받아
각 사분면과 축 위 점 개수를 집계하는 문제입니다.

## 코드

```python
q=[0]*5
for i in range(int(input())):
    x,y=map(int,input().split())
    if x==0 or y==0:q[0]+=1
    elif x>0 and y>0:q[1]+=1
    elif x<0 and y>0:q[2]+=1
    elif x<0 and y<0:q[3]+=1
    else:q[4]+=1
for i in range(1,5):
    print('Q%d: %d' %(i,q[i]))
print('AXIS:',q[0])
```

## 설명

조건 분기로 `Q1~Q4`와 `AXIS` 카운트를 증가시키고,
문제 형식대로 출력합니다.
