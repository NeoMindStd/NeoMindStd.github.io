# 무엇인가

XSplit Broadcaster는 Windows에서 방송/녹화를 빠르게 시작하기 좋은 상용 계열 툴입니다. OBS보다 초반 진입이 더 쉽다고 느끼는 사용자가 많고, 장면 구성과 UI 흐름이 직관적인 편입니다.

# 공식 사이트와 다운로드 경로

- 공식 사이트: [XSplit Broadcaster](https://www.xsplit.com/)
- 다운로드: [공식 다운로드 페이지](https://www.xsplit.com/broadcaster)

# 지원 OS와 설치 방법

## Windows

공식 다운로드 페이지에서 Windows용 설치 파일이나 패키지를 내려받아 설치합니다. 설치가 끝나면 첫 실행과 버전 확인을 한 번 진행하는 편이 좋습니다. 설치 후 카메라, 마이크, 게임 캡처 권한과 그래픽 API 연동 상태를 먼저 확인하는 편이 좋습니다.

# 기본 사용법

1. 새 장면을 만들고 게임, 웹캠, 이미지, 텍스트 같은 소스를 추가합니다.
2. 오디오와 비디오 설정에서 해상도, 프레임, 입력 장치 레벨을 먼저 맞춥니다.
3. 짧은 로컬 녹화나 테스트 송출로 품질을 확인한 뒤 실제 방송을 시작합니다.

# 주의할 점

기능 범위를 넓힐수록 시스템 자원 사용량을 더 의식해야 합니다. 특히 게임 방송에서는 GPU 인코더와 오버레이 소스 수가 성능에 영향을 줍니다.

# 요약

XSplit Broadcaster는 Windows에서 방송 화면을 빠르게 꾸미기 좋은 도구입니다. 먼저 짧은 테스트 녹화로 시스템 부담과 음량 밸런스를 확인하는 흐름이 안정적입니다.
