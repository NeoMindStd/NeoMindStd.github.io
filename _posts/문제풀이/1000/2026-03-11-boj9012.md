# 풀이

## 문제

괄호 문자열이 VPS(올바른 괄호 문자열)인지
판별하는 문제입니다.

## 코드

```python
for _ in range(int(input())):
    string = input()
    stack = []
    push = stack.append
    pop = stack.pop
    flag = True
    for char in string:
        if char == '(':
            push(char)
        else:
            if not stack:
                flag = False
                break
            pop()
    print('YES' if not stack and flag else 'NO')
```

## 설명

`(`는 push, `)`는 pop하며 검사하고,
중간에 비어 있는데 pop이 필요하거나 끝났을 때 스택이 남아 있으면 `NO`입니다.
