# 풀이

## 문제

알파벳 문자열을 전화 다이얼 규칙에 따라
숫자로 바꿀 때 걸리는 총 시간을 구하는 문제입니다.

## 코드

```python
time = 0
for ch in input():
    pos = ord(ch)
    if ord(ch) > ord('R'):
       pos-=1
    if ord(ch) > ord('W'):
       pos-=1
    time += 3+((pos-ord('A'))//3)
print(time)
```

## 설명

`S`, `Z`가 포함된 구간 보정을 반영한 뒤,
문자 위치를 이용해 다이얼 소요 시간을 누적합니다.
