# 풀이

## 문제

그릇 배열 문자열에서
같은 방향이 연속이면 5, 바뀌면 10 높이를 더하는 문제입니다.

## 코드

```python
bowls = input()

h, c = 0, None
for bowl in bowls:
    if c==bowl: h+= 5
    else: h+= 10
    c = bowl
    
print(h)
```

## 설명

직전 문자와 현재 문자를 비교해
높이 증가량을 누적합니다.
