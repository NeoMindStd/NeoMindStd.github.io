# 풀이

## 문제

세 변 길이가 주어질 때
직각삼각형이면 `right`, 아니면 `wrong`을 출력하는 문제입니다.

## 코드

```python
while True:
    a, b, c = list(map(int, input().split()))
    if(a + b + c == 0): break
    print("right" if a**2 == b**2 + c**2 else ("right" if b**2 == a**2 + c**2 else ("right" if c**2 == a**2 + b**2 else "wrong")))
```

## 설명

세 변 중 어떤 변이 빗변인지 모르므로
세 경우의 제곱합 관계를 모두 검사해 판정합니다.
