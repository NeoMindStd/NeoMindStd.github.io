# 풀이

## 문제

2진수 문자열을 8진수 문자열로 변환하는 문제입니다.

## 코드

```python
print(format(int(input(),2),'o'))
```

## 설명

입력 2진수를 정수로 변환한 뒤
`format(..., 'o')`로 8진수 문자열을 출력하면 됩니다.
