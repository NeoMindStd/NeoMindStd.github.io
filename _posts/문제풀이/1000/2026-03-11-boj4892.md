# 풀이

## 문제

주어진 연산 과정을 따랐을 때 결과의 홀짝과 최종 수를 출력하는 문제입니다.

## 코드

```python
T=1
while True:
    a=int(input())*3
    if a==0:break
    f=a%2==0
    b=(a//2 if f else (a+1)//2)*3//9
    print(f'{T}. {"even"if f else"odd"} {b}')
    T+=1
```

## 설명

문제에서 제시한 계산 절차를 그대로 구현하고
중간 값의 홀짝 여부에 따라 `even` 또는 `odd`를 출력합니다.
