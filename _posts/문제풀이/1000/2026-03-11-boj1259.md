# 풀이
## 문제

숫자 문자열이 앞뒤로 같은지(팰린드롬) 판별하는 문제입니다.
입력 `0`이 나오면 종료합니다.

## 코드

```python
while True:
    n=list(input())
    if n[0]=='0':break
    f=True
    for i in range(len(n)//2):
        if n[i]!=n[-i-1]:
            f=False
            break
    print('yes'if f else'no')
```

## 설명

문자열의 앞쪽 절반만 순회하면서
대칭 위치 문자(`i`, `-i-1`)를 비교합니다.

하나라도 다르면 `no`, 끝까지 같으면 `yes`를 출력합니다.
