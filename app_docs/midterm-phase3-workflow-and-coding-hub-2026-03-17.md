# 중기계획 3차 실행: 워크플로우 허브와 코딩 도구 허브 강화

## 목적

- 실무용 AI 허브를 개념 모음이 아니라 선택형 허브로 바꾸기
- AI 코딩 도구 허브에서 설치 난이도와 사용 성격이 바로 보이게 만들기

## 반영 내용

### 실무용 AI·워크플로우 허브

- `업무형 / 리서치형 / 에이전트형 / 설계형` 비교표 추가
- 업무 성격에 따라 어떤 글부터 읽어야 하는지 추천 순서 추가

### AI 코딩 도구 허브

- `설치 난이도 / 작업 성격 / 로컬-원격 감각` 비교표 추가
- `쉬운 진입 / 터미널 중심 / IDE 중심 / 개념 먼저`로 나눈 추천 순서 보강

## 기대 효과

1. 허브에서 바로 이탈하지 않고, 내 상황에 맞는 다음 글을 더 빠르게 고르게 됨
2. AI 허브군이 검색에 잡히기 시작했을 때 snippet 가치와 체류시간을 함께 올릴 가능성
3. 이후 Search Console에서 클러스터 허브별 노출이 생기면 title/description을 더 정교하게 조정하기 쉬움

## 다음 후보

1. 생성형 창작 AI 허브에도 `취미형 / 썸네일형 / 상업형` 구분 추가
2. 각 허브 상단에 대표 카드형 링크나 썸네일 강조 추가
3. Search Console 재수집 후 허브별 CTR 낮은 곳부터 메타 2차 조정
