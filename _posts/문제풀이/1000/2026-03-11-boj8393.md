# 풀이

## 문제

자연수 `n`이 주어질 때
`1`부터 `n`까지의 합을 구하는 문제입니다.

## 코드

```python
a = int(input())

aSum = 0
for i in range(1, a+1):
    aSum += i

print(aSum)
```

## 설명

반복문으로 `1..n`을 누적 합산해 출력합니다.
