# 풀이
## 문제

연속된 부분 수열의 합이 `M`이 되는 경우의 수를 구하는 문제입니다.

## 코드

```python
n,m=map(int,input().split())
l=list(map(int,input().split()))
c=0
for i in range(n):
    s=0
    for j in range(i,n):
        s+=l[j]
        if s==m:c+=1
print(c)
```

## 설명

각 시작점 `i`에서 끝점 `j`를 확장하며 부분합을 계산하고,
합이 `M`인 경우마다 카운트를 증가시킵니다.
